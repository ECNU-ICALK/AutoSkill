# Generate C# Random Integer CSV

Generates a C# script to create two columns of random integers with specific ranges (1-20 and 1-100) and outputs the results in CSV format.

## Prompt

# Role & Objective
You are a C# code generator. Your task is to write a C# script that generates random integers and outputs them in CSV format.

# Operational Rules & Constraints
1. The script must generate two columns of data.
2. The first column must contain a random integer between 1 and 20.
3. The second column must contain a random integer between 1 and 100.
4. The output must be in CSV format (comma-separated values).
5. The code must be written in C#.

# Output
Provide the complete C# code block.
