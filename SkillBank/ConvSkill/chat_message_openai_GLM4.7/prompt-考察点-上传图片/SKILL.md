# prompt / 考察点 / 上传图片

General SOP for common requests related to prompt, 考察点, 上传图片.

## Prompt

Follow this SOP (replace specifics with placeholders like <PROJECT>/<ENV>/<VERSION>):
1) 视觉-语言对齐的精细度（“眼脑”协调测试
2) 目的是测试模型是否真的“看懂”了，还是只是在猜概率
3) 1. 密集与遮挡物体识别
4) Prompt（上传图片）：** 找一张冰箱里塞满食物的照片，或者乱糟糟的办公桌照片
5) 提问：** “请帮我列出图片里所有的饮料品牌。另外，我的钥匙在哪里
6) 考察点：** OCR（光学字符识别）能力，小目标检测能力，以及空间方位感（“在苹果左边” vs “在桌子上
7) 2. 梗图与隐喻理解（多模态思维链
8) Prompt（上传图片）：** 找一张网络热门梗图（Meme），最好是带有文字和讽刺意味的
9) 提问：** “这张图好笑在哪里
10) 考察点：** 模型能否理解图片背后的文化隐喻、幽默感以及图文结合产生的化学反应，而不仅仅是描述画面

For each step, include: action, checks, and failure rollback/fallback plan.
Output format: for each step number, provide status/result and what to do next.
