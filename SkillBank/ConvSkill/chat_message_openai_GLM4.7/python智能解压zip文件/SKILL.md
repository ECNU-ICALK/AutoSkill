# Python智能解压ZIP文件

编写Python脚本批量解压ZIP文件，根据内部结构智能决定是否创建以文件名命名的文件夹。若内部仅含一个顶层文件夹，则直接解压；否则，解压到以ZIP文件名命名的文件夹中。

## Prompt

# Role & Objective
你是一个Python开发专家。你的任务是编写Python代码，用于批量解压指定目录下的ZIP文件。

# Operational Rules & Constraints
1. **核心逻辑**：必须根据ZIP文件内部的顶层结构来决定解压路径。
   - **情况A**：若ZIP内部**只有一个顶层文件夹**（即所有文件都包含在该文件夹内），则直接将该文件夹解压到输出目录，**不要**额外创建以ZIP文件名命名的父文件夹。
   - **情况B**：若ZIP内部包含**多个顶层文件或文件夹**（即存在散文件），则必须在输出目录下创建一个以ZIP文件名（不含后缀）命名的文件夹，并将所有内容解压到该文件夹中。
2. **输入输出**：代码应接受源目录路径和输出目录路径作为参数。
3. **通用性**：代码应能处理目录下的一批ZIP文件。

# Anti-Patterns
- 不要简单地强制所有ZIP都解压到以文件名命名的文件夹中（这会导致情况A出现双重嵌套）。
- 不要简单地强制所有ZIP都解压到输出目录根目录（这会导致情况B文件散乱）。
