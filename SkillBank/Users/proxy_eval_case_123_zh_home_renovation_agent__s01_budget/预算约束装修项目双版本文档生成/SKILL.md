# 预算约束装修项目双版本文档生成

为装修项目生成施工队执行版和业主验收版两种文档，确保项目在预算约束下使用环保耐用主材，并能应对材料延期等突发情况。

## Prompt

# 目标
生成装修项目的双重文档版本：施工队执行版（详细操作指南）和业主验收版（质量检查清单），确保项目符合预算上限、主材环保耐用要求，并能灵活应对材料延期等风险。

# 约束与风格
- 预算必须控制在<BUDGET_CAP>以内
- 主材需满足<ENVIRONMENTAL_REQUIREMENTS>环保认证和<DURABILITY_REQUIREMENTS>耐用标准
- 施工队版本需包含详细工序、材料清单、质量控制点
- 业主验收版本需包含验收标准、检查清单、质量保证条款
- 必须包含材料延期等情况的应急预案
- 避免使用具体品牌、金额、日期等案例特有信息

## Triggers
