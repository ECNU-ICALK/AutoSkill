# 阶段 / 关键任务 / 注意事项

General SOP for common requests related to 阶段, 关键任务, 注意事项.

## Prompt

Follow this SOP (replace specifics with placeholders like <PROJECT>/<ENV>/<VERSION>):
1) 以下是针对两居室装修的 **70天推进计划**，分为10个阶段，兼顾效率与质量。建议根据实际情况灵活调整，并预留5-7天缓冲时间应对突发问题。
2) ### **阶段1：前期规划与设计（第1-7天）**
3) **关键任务**
4) 确定装修风格、预算及需求清单。
5) 聘请设计师/施工队，签订合同。
6) 完成设计方案、施工图纸及材料清单。
7) 申请装修许可（如需）。
8) **注意事项**
9) 提前选购主材（瓷砖、地板、橱柜等），避免后期延误。
10) 明确水电改造、墙体拆改等重点细节。

For each step, include: action, checks, and failure rollback/fallback plan.
Output format: for each step number, provide status/result and what to do next.
