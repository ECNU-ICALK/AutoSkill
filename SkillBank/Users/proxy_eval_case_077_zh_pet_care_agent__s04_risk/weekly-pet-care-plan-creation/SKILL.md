# weekly-pet-care-plan-creation

Creates a structured weekly care plan for dogs with specified core components.

## Prompt

# Goal
Generate a comprehensive weekly care plan for dogs that includes feeding schedules, exercise routines, grooming, deworming schedules, and weight tracking.

# Constraints & Style
- Must include five core components: feeding plan, walking/exercise schedule, grooming routine, deworming/vaccination tracking, and weekly weight monitoring
- Structure content by day of week with clear actionable items
- Provide adjustable parameters (frequency, timing, portions) for different dog ages and breeds
- Include health monitoring checkpoints and emergency indicators
- Keep language clear and practical for daily implementation
- Avoid generic pet care lists; focus on the five specified components

# Workflow
1. Identify dog's age, breed, and current health status
2. Create daily feeding schedule with portion sizes and timing
3. Design daily/weekly exercise routine with duration and type
4. Outline grooming tasks with frequency (brushing, bathing, nail trimming)
5. Establish deworming and vaccination calendar
6. Add weight tracking template and monitoring guidelines
7. Include health observation checklist
8. Format as weekly calendar schedule
