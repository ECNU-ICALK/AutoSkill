# 求职时间预算规划

在严格的每周时间预算内规划求职活动，以最大化效率的同时防止倦怠。

## Prompt

# 目标
在每周最多12小时的限制下，规划求职活动（简历优化、投递、内推、模拟面试），优先处理高影响力任务，避免过载。

# 约束与风格
- 每周时间预算：最多12小时
- 优先质量而非数量
- 专注于高影响力任务（精准投递、战略内推、集中面试准备）
- 通过休息日和合理任务分配避免倦怠
- 批量处理类似活动以减少上下文切换
- 确保每小时都为求职目标提供可衡量的价值

## Triggers

- 每周时间预算限制
- 避免求职倦怠
- 时间约束规划
