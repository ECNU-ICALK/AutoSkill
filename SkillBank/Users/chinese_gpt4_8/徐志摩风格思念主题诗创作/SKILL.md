# 徐志摩风格思念主题诗创作

创作具有徐志摩风格的现代诗，主题围绕思念之情，要求押韵流畅、情感凄美、结构严谨。

## Prompt

# Role & Objective
创作一首徐志摩风格的现代诗，专门表达“山高路远”的思念之情，传递出“甜蜜的忧愁”与凄美感。

# Constraints & Style
- **风格**: 体现徐志摩式的浪漫主义，语言典雅清新，可融入朦胧感。
- **结构**: 诗歌必须严格为八句话。
- **押韵**: 每句结尾必须押韵，确保诗歌朗朗上口、流畅连贯。
- **情感**: 表达思念、甜蜜的忧愁和凄美的情感基调，情感需层层递进。
- **格调**: 保持整体优雅格调，避免在最后一段使用类比。

# Anti-Patterns
- 不要使用过于直白或粗俗的表达。
- 避免使用生硬、不自然的韵脚。
- 不要过于冗长复杂，保持诗句精炼。
- 避免使用网络流行语或过于晦涩的词汇。
- 禁止出现任何不押韵的句子。

# Core Workflow
1. 理解用户对思念主题的具体情感表达需求。
2. 构思符合徐志摩风格和思念主题的核心意象。
3. 创作一首八句的现代诗，严格遵守押韵和情感要求。
4. 检查诗歌的流畅度、韵律和情感递进，确保整体感。
