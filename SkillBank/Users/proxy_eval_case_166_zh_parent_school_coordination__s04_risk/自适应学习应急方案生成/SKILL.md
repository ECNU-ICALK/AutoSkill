# 自适应学习应急方案生成

生成在表现不佳、身体不适、倦怠或其他不利条件下既能减轻负荷又能保持连续性和学习动力的自适应学习应急方案。确保即使在条件不佳时教育参与也不会完全中断。

## Prompt

# 目标
生成在表现不佳、身体不适、倦怠或其他不利条件下既能减轻负荷又能保持连续性和学习动力的自适应学习应急方案。确保即使在条件不佳时教育参与也不会完全中断。

# 约束与风格
- 必须包含明确的激活触发条件（例如：表现低于阈值、健康问题、倦怠迹象、动机低下）
- 必须指定具体的负荷减轻策略（时长缩短、难度调整、内容简化、频率降低）
- 必须维持核心连续性机制（最低参与要求、关键概念强化、常规维护）
- 必须定义可衡量的恢复正常负荷的标准
- 必须适用于不同学科和活动类型
- 避免完全停止；优先考虑"保持连续性"同时减轻压力
- 提供具体、可操作的减轻水平和连续性要求
- 包含监控和调整检查点
