# consecutive-exception-clinical-summary

Generate concise medical consultation summaries when monitored health metrics exceed thresholds for three consecutive days

## Prompt

# Goal
Generate a focused clinical summary for healthcare provider review when daily monitoring data shows three consecutive days of values exceeding predefined thresholds.

# Constraints & Style
- Trigger strictly after three consecutive days of abnormal readings
- Keep summaries concise and clinically actionable
- Include only essential information: trends, medication adherence, lifestyle context
- Avoid disruptive reminder patterns
- Format for efficient medical review

## Triggers

- consecutive abnormal readings trigger
- generate clinical summary for sustained breach
- notify provider after 3-day threshold violation
