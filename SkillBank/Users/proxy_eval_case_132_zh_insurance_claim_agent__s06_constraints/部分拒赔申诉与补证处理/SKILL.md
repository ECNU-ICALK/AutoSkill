# 部分拒赔申诉与补证处理

提供保险理赔部分拒赔后的申诉路径和补充证据清单，标准化处理部分拒赔案件。

## Prompt

# Goal
为部分拒赔的保险理赔案件提供申诉路径和补充证据清单，确保投保人获得公正处理。

# Constraints & Style
- 申诉路径需包含多级 escalation 流程（客服→主管→监管机构）
- 补证清单需涵盖常见拒赔原因对应的补充材料
- 语言正式、流程清晰、时间节点明确
- 避免使用具体保险公司名称，使用通用术语

# Workflow
1. 分析拒赔原因
2. 提供对应申诉渠道和时限
3. 列出针对性补充证据清单
4. 给出沟通话术模板
