# 宠物护理双版本清单生成

为宠物日常照护和临时代养分别生成结构化清单，确保日常版聚焦常规执行项，代养版强化交接流程与应急规则，包含食欲下降的分级医疗升级路径及行程中断的应急处理分支。

## Prompt

# Goal
生成两套宠物护理清单：1) 日常版用于主人常规照护，2) 代养版用于临时看护人交接，确保格式清晰、内容完整，包含食欲下降的分级医疗升级路径及行程中断的应急处理分支。

# Constraints & Style
- **日常版清单**：聚焦每日/每周固定执行项，使用表格或时间轴展示，包含喂食、运动、清洁、驱虫等核心项目，省略交接流程。
- **代养版清单**：额外包含交接准备、代养执行清单、紧急处理流程、物品清单、健康记录表等模块，使用分级标题和复选框格式。
- **共性要求**：
  - 使用简洁的表格/列表格式，避免长段落
  - 标注关键时间节点和频率（如每日2次、每周1次）
  - 包含紧急联系人和兽医信息
  - 食欲下降需明确分级医疗升级路径（轻度10-20%、中度20-50%、重度>50%食欲减退）
  - 所有内容可导出为打印版或日历同步格式
  - 紧急联系人需建立层级结构并明确升级时限
  - 代养版需明确区分主人指示与看护人指示
  - 包含体重追踪与观察协议
  - 代养版需包含预算应急计划以应对临时看护成本超支
  - 隐去所有宠物专属细节信息

# Workflow
1. 定义日常照护常规流程与预防性监测项
2. 定义临时代养交接流程与物品清单
3. 实施食欲下降的分级医疗升级规则与行动步骤
4. 添加行程取消、延误及预算超支的应急处理分支
5. 验证清单包含明确触发条件与升级时限
