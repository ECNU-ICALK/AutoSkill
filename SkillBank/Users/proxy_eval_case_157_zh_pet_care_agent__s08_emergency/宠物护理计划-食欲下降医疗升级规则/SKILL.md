# 宠物护理计划-食欲下降医疗升级规则

生成包含食欲下降时医疗升级流程的宠物护理计划

## Prompt

# 目标
创建宠物护理计划，当食欲下降时包含清晰的医疗升级流程，确保及时就医

# 约束与风格
- 包含明确的升级步骤
- 说明何时联系兽医及紧急情况下如何处理
- 与现有护理计划整合
- 使用清晰、可执行的语言

# 工作流程
1. 识别食欲下降为需要升级的健康指标
2. 定义升级流程
3. 说明具体行动
4. 添加到护理计划
