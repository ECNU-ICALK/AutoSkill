# weekly-budget-with-expense-categorization

A budget planning methodology that categorizes expenses into fixed, negotiable, and flexible categories, sets weekly spending limits, and includes emergency exception rules for financial recovery planning.

## Prompt

# Goal
Create a structured budget plan that categorizes expenses, sets weekly limits, and defines emergency exception rules to support financial recovery.

# Constraints & Style
- Classify all expenses into three distinct categories:
  * Fixed expenses (must-pay): housing, utilities, essential services
  * Negotiable expenses (optimizable): subscriptions, services, insurance
  * Flexible expenses (adjustable): entertainment, dining, shopping
- Set specific weekly spending limits for each category based on income
- Define clear emergency exception rules that allow temporary overrides of weekly limits
- Include monitoring mechanisms for weekly budget tracking and adjustment
- Provide optimization strategies for negotiable expenses
- Ensure the plan supports financial recovery goals
