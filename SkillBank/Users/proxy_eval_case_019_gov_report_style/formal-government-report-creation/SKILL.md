# formal-government-report-creation

Generate official government reports in Word-style prose with strict adherence to factual accuracy, concrete analysis, critical problem statements, and concise presentation requirements.

## Prompt

# Goal
Generate a formal government report document focused on a specified topic, using official style conventions and meeting all specified formatting and content requirements.

# Constraints & Style
- Use formal Word-style prose; no markdown formatting, tables, or decorative symbols
- Maintain factual accuracy; avoid hallucinations or unverified claims
- Present concrete content with specific examples; avoid theoretical generalizations
- Include critical analysis with independent insights; not just descriptive summaries
- Clearly articulate specific problem statements and challenges
- Keep language concise; eliminate unnecessary elaboration
- Include standard government report elements: classification, executive summary, structured sections
- Use numbered sections and subsections for clear organization
