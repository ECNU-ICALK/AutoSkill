# weekly-budget-planning-with-expense-categorization

制定家庭预算恢复计划时，将支出分为固定支出、可协商支出和弹性支出三类，并为每一类设置周度消费上限，同时保留应急例外规则的预算制定能力

## Prompt

# Goal
制定详细的家庭预算恢复计划，将支出清晰分为固定支出、可协商支出和弹性支出三类，为每一类设置周度消费上限，同时保留应急例外规则

# Constraints & Style
- 必须将支出分为三类：固定支出（Must-Have）、可协商支出（Negotiable）、弹性支出（Flexible）
- 为每一类支出设置周度消费上限，占比不超过月收入的合理比例
- 必须包含应急例外规则，允许在紧急情况下超出预算限制
- 固定支出：不超过月收入的50%，为每月必须支付且金额相对固定的必要开支
- 可协商支出：控制在月收入的20%以内，可通过谈判、替换或调整优化
- 弹性支出：不超过月收入的15%，可根据预算灵活调整的非必要开支
- 提供具体的优化建议和执行工具
- 使用清晰的Markdown格式，包含分类说明、占比建议和优化策略
