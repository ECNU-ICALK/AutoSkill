# 装修隐蔽工程双轨验收规范

为业主和施工队分别提供可执行的隐蔽工程验收标准与动作指引，覆盖拆改、水电、防水三大关键环节，确保责任清晰、动作可追溯、风险可拦截。

## Prompt

# Goal
生成两份平行、对齐、互为印证的隐蔽工程执行与验收文档：一份面向施工队（含明确工艺指令、材料规格、工序顺序、禁止行为）；一份面向业主（含可视化检查方法、合格判定标准、不合格立即叫停动作），二者在关键控制点上严格对应。

# Constraints & Style
- 必须严格基于用户确认的5个生死线检查项（承重结构、水电点位、水管热熔、防水高度、电路绝缘）展开，不新增检查维度；
- 所有技术参数必须与用户原始要求一致：承重墙≥24cm、热熔稳压5秒、防水淋浴区≥1.8m、绝缘电阻≥0.5MΩ（潮湿区≥1.0MΩ）；
- 施工队版使用祈使句+动词开头（如“必须开槽深度=管径+1.5cm”“严禁用胶水粘死PVC线管”），禁用解释性语言；
- 业主版使用“怎么查？→ 合格标准 → 不合格立刻叫停！”三段式结构，全部操作需可在3分钟内完成，无需专业设备（允许使用卷尺、手机慢镜头、激光笔等常见工具）；
- 禁止出现品牌推荐、预算分配、工期调度等跨能力域内容；
- 输出为纯文本双栏对照格式，左栏【施工队执行版】，右栏【业主验收版】，用分隔线清晰隔离。

# Workflow
1. 提取用户定义的5个生死线检查项及其原始判定标准；
2. 为每一项分别撰写施工队侧的刚性工艺指令（含动作、参数、禁止项）；
3. 为同一项同步撰写业主侧的极简自查动作（含工具、步骤、阈值、响应）；
4. 按照‘承重→点位→热熔→防水→绝缘’顺序组织双栏输出；
5. 结尾附加一句通用口诀，凝练双方共同守则（如用户原句‘承重不碰、点位不移、热熔不糊、防水不矮、绝缘不低’）。
