# 装修工期弹性调度-材料延期应对

当主材（如橱柜、瓷砖、地板等）交付延期时，动态重排70天装修计划以保障关键路径不延误、隐蔽工程不受损、验收节点可控的调度方法。

## Prompt

# Goal
在主材实际到货比原计划晚7天的前提下，重新优化两居室（70–90㎡）装修全流程工期安排，确保：① 水电/防水等隐蔽工程不被压缩或跳过；② 各阶段衔接仍满足工艺等待期（如油漆干透、闭水试验时长）；③ 总工期延长≤3天（即≤73天），且关键交付日（如软装进场、竣工验收）可接受微调。

# Constraints & Style
✅ 必须保留全部原计划中强制性工艺节点与时长：
　　- 水电打压稳压≥30分钟、压降≤0.05MPa；
　　- 防水闭水试验≥48小时且需楼下业主签字；
　　- 油漆每遍间隔≥6小时（晴天）、腻子打磨后静置24h再上漆；
　　- 地板铺装前地面含水率≤12%（需实测）。
✅ 主材延期仅影响其对应安装工序起始日，不得倒置工序（如：禁止未铺砖先装橱柜）；
✅ 优先利用原计划中非依赖主材的并行窗口（如：D11–D18水电施工期间，D19–D25泥工可部分前置至D16–D22，若基层已处理完毕）；
✅ 禁止牺牲质量换时间：不缩短养护期、不跳过验收、不合并责任方作业；
✅ 所有调整必须标注‘可缓冲项’（如：开荒保洁、软装摆场、灯具调试）与‘刚性项’（如：防水闭水、电路绝缘测试）。

# Workflow
1. 识别延期主材类型及原定安装时段（例：橱柜原D36–D38安装 → 延至D43–D45）；
2. 向后平移其直接依赖工序（如：全屋定制柜原D43–D50 → 改为D50–D57）；
3. 前置非依赖工序填补空档（例：D31–D35油漆完成后，D36–D42原定安装时段若空闲，可提前开展吊顶检修口封板、背景墙饰面收口等无主材依赖工作）；
4. 将原计划中柔性环节（如：D51–D55细部修补、D56–D60开荒保洁）压缩1–2天，并将节省时间注入关键路径缓冲区；
5. 输出更新版甘特图，标出新关键路径与各阶段浮动时间（Total Float）。
