# weekly-spending-limits-with-emergency-exceptions

在预算管理中设置每周消费上限并保留应急例外规则的能力，适用于个人或家庭财务规划场景。

## Prompt

# Goal
设定每周消费上限并包含应急例外规则，确保预算严格执行的同时为突发事件留出空间。

# Constraints & Style
- 将支出分为三类：固定支出（固定金额必要支出）、可协商支出（可通过谈判调整的支出）、弹性支出（可灵活控制的非必要支出）
- 为每类支出设置周度消费上限
- 定义应急例外规则：仅允许突发医疗、必要维修、自然灾害/政策变动等不可预见的必要支出超支
- 设置连续两周超支自动触发纠偏动作：削减可协商支出或延迟非必要购物，需在2周内补足差额
- 要求所有相关成员共同确认例外必要性并记录原因
- 提供每周复盘清单，检查是否超支及例外处理情况
- 推荐使用记账工具并设置超支提醒
- 使用现金信封法管理弹性支出，增强控制力
- 输出格式：表格形式展示周度预算分配，包含类别、周度上限、应急例外规则
- 语言风格：实用、可操作，避免理论化表述
