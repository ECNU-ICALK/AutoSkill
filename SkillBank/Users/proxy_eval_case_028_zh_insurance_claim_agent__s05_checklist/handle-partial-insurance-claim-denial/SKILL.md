# handle-partial-insurance-claim-denial

提供保险理赔部分拒赔后的申诉路径和补充证据清单，指导用户依法依规处理拒赔争议并完成二次申诉。

## Prompt

# Goal
处理保险理赔部分拒赔事件，提供明确的申诉流程、所需补充证据清单及法律救济途径，确保用户权益通过合规渠道获得保障。

# Constraints & Style
- 必须包含三个层级的申诉路径：保险公司内部申诉、监管机构投诉、司法救济途径
- 补充证据清单需按优先级排序，区分必要证据和可选证据
- 必须引用《保险法》等通用法规条款作为依据，但不可指定具体保险公司
- 语言需保持中立客观，避免情绪化表述，聚焦可执行操作
- 提供标准化申诉信模板框架（含必填要素），但不填写具体案例信息
- 明确时效要求（如内部申诉通常需10-15工作日响应）

# Workflow
1. 接收拒赔通知后，立即书面确认拒赔理由和金额
2. 评估拒赔合理性，准备补充证据材料
3. 按优先级启动申诉流程：
   - 第一阶段：向保险公司提交书面申诉及补充材料
   - 第二阶段：若未获满意回应，向银保监会/保险行业协会投诉
   - 第三阶段：必要时通过仲裁或诉讼解决
4. 持续跟进并记录各阶段处理进展
