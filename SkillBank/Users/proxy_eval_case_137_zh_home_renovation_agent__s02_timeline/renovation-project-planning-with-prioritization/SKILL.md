# renovation-project-planning-with-prioritization

创建两居室装修70天推进计划，优先安排拆改和水电工程，对隐蔽工程实施重点把控，同时遵守预算约束并指定环保耐用主材。

## Prompt

# Goal
制定两居室装修70天推进计划，明确各阶段时间分配、任务清单和关键控制点，确保拆改和水电工程优先执行，隐蔽工程得到重点把控，同时遵守预算约束并指定环保耐用主材。

# Constraints & Style
- 必须包含七个核心阶段：前期准备、拆除与结构改造、水电与隐蔽工程、泥木与基层施工、油漆与表面处理、主材安装、收尾与保洁
- 首要安排拆除与改造阶段（第1-14天）和水电与隐蔽工程阶段（第15-28天），这两个阶段必须前置
- 对隐蔽工程实施重点把控，包括材料验证（品牌、规格、证书）、工艺标准（安装技术、间距要求）、测试规程（压力测试、电路测试）、文档要求（照片记录、签收确认）、三方验收等具体措施
- 每个阶段需标注时间范围、具体任务、关键点和注意事项
- 需提供简化甘特图展示时间分配
- 保留缓冲时间（5-7天）应对突发情况
- 明确交叉施工可能性和每日巡查要求
- 遵守指定预算约束
- 指定环保、耐用的主材要求
- 使用中性、专业语言，包含清晰技术规格

# Workflow
1. 解析用户需求，确认装修面积、周期和预算要求
2. 按照优先级分配时间：拆改→水电→泥木→油漆→安装→收尾
3. 为每个阶段设计任务清单和验收标准
4. 标注关键控制点，特别是隐蔽工程的质量把控措施
5. 生成时间分配表和甘特图可视化
6. 添加注意事项和缓冲策略
