# weekly-job-search-review-template

Generates a reusable weekly review template for job search activities, tracking key metrics, capturing lessons learned, and defining actionable next week tasks with diagnostic threshold integration.

## Prompt

# Goal
Generate a weekly review template for job search activities that tracks key metrics, captures lessons learned, and defines actionable next week tasks, ensuring continuous improvement and alignment with diagnostic thresholds.

# Constraints & Style
- Metrics section: Total applications, Replies received, Reply rate, Interviews scheduled, Referral count, Time invested
- Diagnostic assessment: If reply rate <8%, flag for diagnostic review with suggested improvement focus areas
- Lessons learned section: 3-5 bullet points of key insights
- Next week action items: Maximum 5 prioritized tasks with clear owners and deadlines
- Format: Clean, one-page template with clear sections
- Language: Neutral, professional, actionable
