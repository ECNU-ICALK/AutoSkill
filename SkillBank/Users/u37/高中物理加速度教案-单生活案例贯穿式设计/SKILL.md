# 高中物理加速度教案-单生活案例贯穿式设计

一种面向高中物理教学的教案生成能力，要求以单一真实、可感知的生活案例（如电梯启停）为线索，贯穿教学目标、重难点、全过程环节、板书与反思，禁用表格与LaTeX渲染符号，适配Word文档排版习惯。

## Prompt

# Goal
生成一份符合新课标要求的高中物理《加速度》教案，严格满足以下约束：
- 全程以**一个用户指定或典型的生活案例**（如“电梯启动与制动时人体超重/失重感”）为统一主线；
- 该案例须自然融入情境导入、概念建构、辨析深化、应用迁移、小结作业各教学环节，不新增其他案例；
- 输出格式为纯文本Word友好结构：禁用Markdown表格、禁用代码块、禁用LaTeX渲染符号（如用'Δv/Δt'代替'\( a = \frac{\Delta v}{\Delta t} \)'），公式使用单$行内格式（如'$a = \frac{\Delta v}{\Delta t}$'）；
- 语言简洁、段落分明、标题层级清晰（如'一、教学目标'→'1. 物理观念'），适配直接粘贴至Word使用。

# Constraints & Style
- 必须体现加速度的三大核心认知：① 描述速度变化的快慢（非大小）；② 矢量性（方向由Δv决定，可与v同向或反向）；③ v-t图像中斜率即a；
- 所有教学活动设计须围绕同一案例展开推演（例如：用电梯速度变化数据计算a，画其v-t草图，分析超失重成因，关联牛顿第二定律）；
- 禁用任何表格、图表占位符、多列布局；所有对比、步骤、列表均用编号段落或'•'符号呈现；
- 明确设定正方向（如“向上为正”），所有符号运算与方向判断以此为基准；
- 强调关键概念区分：'速度vs.速率'、'Δv vs. v'、'a≠0不等于正在加速'；
- 每个教学阶段（启动/匀速/制动）须含自洽定量计算（如a=Δv/Δt），数值符合现实典型量级（如电梯加速度≈±0.5–2.0 m/s²）；
- 板书设计以纯文本缩进+符号模拟（如'→'表示逻辑指向，'★'强调要点）；
- 教学反思聚焦该案例在突破迷思概念（如'a与v方向关系'）中的实效性；
- 不得出现'例如''比如''类似地'等泛化表述——每个教学行为必须明确绑定该案例。

# Workflow
1. 确认并锚定唯一生活案例（若用户未指定，则默认采用'电梯运行中的超重与失重现象'）；
2. 基于该案例反向推导教学目标（物理观念/科学思维/科学探究/态度责任四维度）；
3. 按'一、教学目标 → 二、教学重难点 → 三、教学过程（分环节描述教师与学生活动） → 四、板书设计 → 五、教学反思建议'顺序组织内容；
4. 每个环节中，所有举例、计算、提问、讨论均仅调用该案例的变量（如电梯初速、末速、时间、受力变化）；
5. 输出前检查：全文无表格、无LaTeX渲染、无跨案例跳跃、无抽象泛指，全部可直接复制到Word并保持可读性。
