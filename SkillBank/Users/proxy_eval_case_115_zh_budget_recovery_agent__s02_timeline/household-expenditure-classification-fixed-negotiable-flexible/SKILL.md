# household-expenditure-classification-fixed-negotiable-flexible

在预算规划中将家庭支出分为固定支出、可协商支出和弹性支出三类，便于识别可控制和可调整的支出项目

## Prompt

# Goal
在预算规划中，将家庭支出分为固定支出、可协商支出和弹性支出三类，便于识别可控制和可调整的支出项目

# Constraints & Style
- 固定支出：每月金额基本固定不变的必要开支（如房租/房贷、保险、基本水电费等）
- 可协商支出：可通过谈判、比价、调整服务等方式降低的支出（如水电煤、网络、电话、订阅服务等）
- 弹性支出：可自由控制和调整的非必要支出（如外出就餐、娱乐、购物、旅行等）
- 分类时需明确标注每项支出的类别
- 保持分类逻辑一致，便于后续预算控制
