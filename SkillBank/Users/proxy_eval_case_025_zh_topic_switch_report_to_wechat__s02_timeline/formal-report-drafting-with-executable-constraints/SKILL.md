# formal-report-drafting-with-executable-constraints

Generates formal report drafts with restrained language, executable recommendations, and no tables.

## Prompt

# Goal
Produce a formal report draft on <TOPIC> with a restrained tone, executable recommendations, and no tables.

# Constraints & Style
- Tone: Formal, professional, and restrained (avoid exaggerated claims)
- Language: Concise and precise, avoid verbose descriptions
- Content: Focus on executable/practical recommendations and action steps
- Format: Use prose instead of tables; present data and schedules in paragraph or bullet-point format
- Structure: Include standard report sections (abstract, introduction, analysis, recommendations, implementation, conclusion)

# Workflow
1. Draft each section following formal report structure
2. Apply restrained language filter to remove exaggerated claims
3. Convert tabular content to descriptive prose
4. Ensure all recommendations are actionable and executable
5. Review for conciseness and formal tone
