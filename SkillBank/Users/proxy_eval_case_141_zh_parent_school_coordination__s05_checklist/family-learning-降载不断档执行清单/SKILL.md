# family-learning-降载不断档执行清单

为家庭学习支持设计的可打印/手写式弹性降载执行工具：当孩子或家长状态波动时，基于三级能量信号（轻度疲态／中度倦怠／明显低能量）自动启用对应精简任务组合，维持数学逻辑感、阅读眼动节奏、英语语音流三大核心通路，零准备、零决策、不中断学习联结。

## Prompt

# Goal
生成一份开箱即用、适配打印（A5/A6）或手写填写的《家庭学习降载不断档执行清单》，支持按日启用三级降载模式（Level 1–3），在孩子/家长状态下降时维持三大核心学习通路（数学逻辑感、阅读眼动节奏、英语语音流），无需额外配置、不引入外部资源、不扩展用户未确认动作。

# Constraints & Style
- 输出为纯文本 Markdown 表格格式，仅含必要字段：日期｜状态信号（三选一：轻度疲态／中度倦怠／明显低能量）｜启用级别（单选）｜周中任务（1.5h）｜周末任务（周六／周日分栏，3h总长）；
- 所有任务描述必须严格复用用户历史明确认可的表述（如‘讲题不写过程’‘听读+指读’‘拍手打节奏’‘积木摆算式’‘关键词接龙’‘能量确认’），禁用未采纳术语（如‘神经通路’‘多感官输入’）、未授权资源（如APP、扫码微课）或扩展动作（如校医对接、正念音频）；
- 每项任务以中性动词开头（无主语），≤12字，禁止‘鼓励’‘坚持’‘克服’等隐含道德压力词汇，改用‘陪一陪’‘翻一翻’‘听一听’‘指一指’‘摆一摆’‘接一接’等；
- Level 3 任务严格限定为：1项「能量确认」+ 自然发生场景提示（不设任务项，仅举例）；
- 禁止出现教师响应、同步话术、设计说明、温馨提示、备选方案或解释性文字；仅输出家庭端可直接勾选执行的动作；
- 周中与周末任务均按学科分栏（数学｜语文｜英语），每科动作明确标注时长与交付形式（如‘录音讲思路’‘指读标3个最长词’）；
- 所有活动满足：① 可立即启动（无需打印/下载/找材料），② 支持亲子共在但不强制互动，③ 有自然结束信号（如1首歌时长、1杯水喝完）；
- 禁止书面作业、计时压力、正确率要求、新知识输入；每科单次活动≤15分钟，总时长符合Level设定（Level 1: 45min，Level 2: 30min，Level 3: ≤10min+自然场景）。

# Workflow
1. 用户填写【日期】与【状态信号】（三选一：轻度疲态／中度倦怠／明显低能量）；
2. 系统自动锁定对应Level并展开当日全部任务；
3. 家长按行执行，完成即勾选✅。
