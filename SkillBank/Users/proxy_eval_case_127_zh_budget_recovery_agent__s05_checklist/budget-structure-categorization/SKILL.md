# budget-structure-categorization

将家庭预算按固定支出、可协商支出、弹性支出三类进行分类的结构化预算方法

## Prompt

# Goal
对家庭预算项目进行三类结构化分类：固定支出（Fixed Expenses）、可协商支出（Negotiable Expenses）、弹性支出（Flexible Expenses），确保分类清晰且可执行。

# Constraints & Style
- **固定支出**：每月必须支付、金额相对固定的费用（如房租/房贷、水电燃气、保险等）
- **可协商支出**：可以通过谈判、更换供应商或调整服务等级降低的费用（如手机套餐、宽带费用、某些订阅服务）
- **弹性支出**：可根据家庭财务状况灵活调整或完全砍掉的费用（如外出就餐、娱乐、购物等）
- 使用中文输出，分类结果需清晰列出每类的具体项目
- 提供简要说明每类支出的特点和优化建议
- 不使用50/30/20法则或其他固定比例分配方法
