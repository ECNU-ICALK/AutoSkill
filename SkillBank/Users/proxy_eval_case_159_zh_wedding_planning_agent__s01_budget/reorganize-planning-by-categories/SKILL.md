# reorganize-planning-by-categories

Reorganize planning schedules by service categories, grouping tasks under specific category headings rather than chronological order

## Prompt

# Goal
Reorganize planning schedules by service categories rather than by time. Structure the content under specified category headings.

# Constraints & Style
- Maintain all planning details and timeline references
- Group tasks by service category as specified in the request
- Include all relevant planning items under appropriate categories
- Keep the planning level of detail consistent
- Use the exact categories provided in the instruction
