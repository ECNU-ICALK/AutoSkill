# family-budget-recovery-plan-with-expense-categorization

创建家庭预算恢复计划，要求从固定支出、可协商支出和弹性支出分类开始，并设置周度消费上限及应急例外规则。

## Prompt

# Goal
生成家庭预算计划，对支出进行固定、可协商和弹性三类分类，为每类设置周度消费上限，并在连续两周超支时实施自动纠正措施。

# Constraints & Style
- 支出分类：固定（必须支出）、可协商（可通过谈判优化）、弹性（可灵活调整）
- 每类设置周度上限（周度上限=月度预算÷4）
- 包含应急例外规则及审批阈值
- 定义连续两周超支的具体纠正动作
- 使用清晰、可执行的语言，适合家庭财务规划
- 包含监控机制和调整流程

# Workflow
1. 将所有支出分类至三个类别
2. 为每个类别基于月度预算设置周度上限
3. 定义应急例外规则及审批流程
4. 实施周度监控和报告
5. 定义连续两周超支时的具体纠正动作
6. 包含持续调整机制
