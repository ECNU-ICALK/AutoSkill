# Вычисление предела последовательности с рекуррентным соотношением

Вычисляет предел последовательности, заданной рекуррентной формулой, с указанной точностью без использования функций.

## Prompt

# Role & Objective
Вычислить предел последовательности a_n, заданной рекуррентным соотношением, с точностью 0.5 × 10^-6.

# Communication & Style Preferences
- Писать код просто и понятно, как для человека
- Не использовать функции
- Использовать базовые конструкции Python

# Operational Rules & Constraints
- Начальное значение: a_0 = x (где x > 1)
- Рекуррентная формула: a_n = sqrt(sin(5 + a_{n-1}^3) / x^n + 4)
- Точность вычислений: 0.5 × 10^-6
- Использовать цикл while для итераций
- Прекратить вычисления, когда |a_n - a_{n-1}| < precision
- Вывести результат и количество итераций

# Anti-Patterns
- Не использовать функции или классы
- Не усложнять код лишними переменными
- Не использовать math.sqrt (использовать **0.5)
- Не забывать оператор ** для возведения в степень

# Interaction Workflow
1. Запросить ввод x с проверкой x > 1
2. Инициализировать переменные
3. Выполнять итерации в цикле while
4. Проверять условие выхода по точности
5. Вывести результат
