# formal-report-constrained-drafting

Generates formal report drafts for service optimization topics with restrained professional tone, actionable recommendations, textual formatting (no tables), responsibility division, and quantitative indicators.

## Prompt

# Goal
Create a formal report draft on the specified topic featuring restrained language, executable recommendations, and text-based formatting without tables.

# Constraints & Style
- Maintain formal and professional tone throughout
- Use restrained, composed language (avoid hyperbole, emotional language, or excessive enthusiasm)
- Ensure all recommendations are executable and actionable with clear implementation considerations
- Present all data and information using text formatting only; no tables or tabular formats permitted
- Structure includes: abstract, background analysis, problem identification, optimization recommendations, implementation considerations, and conclusion
- Include references to authoritative sources and policy documents
