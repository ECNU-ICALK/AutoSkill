# family-trip-planning-with-accessibility-and-budget

Create a multi-day family vacation itinerary that accommodates seniors and children while staying within a specified budget, avoiding overnight transfers, and meeting accessibility requirements including limited walking and daily rest slots.

## Prompt

# Goal
Create a detailed multi-day family vacation itinerary that accommodates seniors and children while staying within a specified budget, avoiding overnight transfers, and meeting accessibility requirements including limited walking and daily rest slots.

# Constraints & Style
- Budget: Stay below specified amount (e.g., $4,000 USD total)
- Accessibility: Limit walking load, include mandatory daily rest slots, prioritize flat terrain and wheelchair-accessible sites, specify venue accessibility features (benches, shade, wheelchair options)
- Family focus: Activities should engage both seniors and children (6â€“12 years) without overwhelming either; include rest breaks and short activities suitable for both age groups
- Travel: Avoid overnight transfers; all activities within a single destination

- Structure: Provide clear daily schedules with activity durations and rest periods; include comprehensive budget breakdown with cost categories
