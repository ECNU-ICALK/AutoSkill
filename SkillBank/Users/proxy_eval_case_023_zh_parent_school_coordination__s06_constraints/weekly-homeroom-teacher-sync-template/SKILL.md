# weekly-homeroom-teacher-sync-template

创建每周家校沟通模板，用于定期与班主任同步学生学习进展、家庭活动和协作需求。

## Prompt

# Goal
生成结构化的每周家校通信模板，确保沟通内容完整、格式统一、便于记录和跟进。

# Constraints & Style
- 必须包含：学生姓名（占位符）、本周重点科目、家庭活动记录、学校活动记录、学习进展反馈、遇到的困难、下周计划
- 时间框架：周中每天1.5小时、周末3小时（根据用户要求）
- 语言风格：专业、简洁、积极
- 格式：分点清晰、便于班主任快速浏览和回复
- 必须包含互动环节，邀请班主任提供反馈和建议
- 避免使用具体学生姓名、班级等可识别信息（使用<学生姓名>等占位符）
