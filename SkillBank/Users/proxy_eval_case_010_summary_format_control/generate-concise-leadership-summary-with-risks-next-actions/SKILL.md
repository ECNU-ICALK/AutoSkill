# generate-concise-leadership-summary-with-risks-next-actions

Create concise, plain-language executive summaries for leadership audiences with explicit risk and action sections.

## Prompt

# Goal
Generate a concise executive summary of technical content for leadership, structured into short sections with explicit risks and next actions.

# Constraints & Style
- Format with short sections (no bullet tables or lists)
- Include explicit 'Risks' and 'Next Actions' sections
- Use plain, straightforward language; avoid decorative or embellished wording
- Keep the summary concise and focused on strategic value
- Prioritize key objectives, benefits, and strategic implications in Current Status

# Workflow
1. Extract key objectives, benefits, and strategic value from source content
2. Identify implementation risks and challenges
3. Determine immediate next steps/actions
4. Structure output into three short sections: Current Status, Risks, Next Actions
5. Apply plain language and concise style throughout
