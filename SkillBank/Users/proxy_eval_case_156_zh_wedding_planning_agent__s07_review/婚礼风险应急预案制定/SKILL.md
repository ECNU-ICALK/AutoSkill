# 婚礼风险应急预案制定

为120人规模、预算30万内的婚礼制定可执行的天气和供应商风险应急预案，包括备用方案、资源调配和紧急沟通机制。

## Prompt

# Goal
为120人规模、预算30万内的婚礼制定针对雨天和供应商爽约的可执行应急预案，确保婚礼流程不受不可抗力影响。

# Constraints & Style
- 必须包含雨天应急预案（户外转室内、帐篷租赁、时间调整方案）
- 必须包含供应商爽约预案（备份供应商名单、替代服务方案）
- 必须预留10%（3万）应急资金专项使用
- 必须包含清晰的决策树和时间线（24h/48h/72h前行动）
- 必须提供客诉通知和供应商沟通模板
- 必须与4个月筹备周期和付款节点匹配
- 避免泛泛而谈，提供具体可执行措施

# Workflow
1. 识别高风险场景：户外婚礼天气、关键供应商（场地/餐饮/摄影/DJ）无故失联
2. 制定备用方案：备用室内场地清单、帐篷租赁商、替代供应商名录
3. 资源调配：计算应急成本、确认备用资源可用性
4. 沟通机制：客诉通知模板、供应商应急联系流程
5. 时间线：明确不同倒计时阶段的决策节点和行动项
