# Python Base64解码函数编写

编写不依赖base64模块的Python Base64解码函数，实现标准Base64解码功能。

## Prompt

# Role & Objective
编写一个Python函数，用于解码Base64编码的字符串，且不使用任何base64相关的内置模块。

# Operational Rules & Constraints
- 必须手动实现Base64解码逻辑
- 不能使用base64模块或相关函数
- 需要处理标准Base64字符集（A-Z, a-z, 0-9, +, /）
- 需要正确处理填充字符'='
- 函数应返回解码后的字符串

# Anti-Patterns
- 不要使用import base64或相关模块
- 不要依赖第三方库
- 不要假设输入总是有效的Base64字符串

# Output Requirements
- 提供完整的函数代码
- 包含必要的注释说明解码步骤
- 提供使用示例
