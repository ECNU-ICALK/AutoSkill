# 设计员工工资计算类

根据用户需求设计并实现一个员工类，包含工资录入、税前工资计算、个税计算、实发工资计算及信息显示功能。

## Prompt

# Role & Objective
设计并实现一个员工(Employee)类，用于管理员工信息和工资计算。

# Communication & Style Preferences
使用中文注释和提示信息，代码结构清晰，方法命名符合语义。

# Operational Rules & Constraints
1. 成员变量：编号、姓名、工龄、基础工资、岗位津贴、效益工资
2. 必须实现构造方法(__init__)和析构方法(__del__)
3. 实现工资录入方法(input_salary)，支持交互式输入基础工资、岗位津贴、效益工资
4. 计算应付工资(calc_salary)：基础工资+岗位津贴+效益工资
5. 计算个人所得税(calc_tax)：3500以下免税，超出3500部分按3%缴纳
6. 计算实发工资(calc_actual_salary)：应付工资-个人所得税
7. 显示员工信息(display_info)：编号、姓名、工龄、应付工资、实发工资
8. 生成员工对象并调用显示方法

# Anti-Patterns
- 不要省略任何必需的方法
- 不要修改税率规则(3500免税，超出部分3%)
- 不要遗漏显示信息中的任何字段

# Interaction Workflow
1. 定义Employee类
2. 实现所有必需方法
3. 创建员工对象实例
4. 调用显示方法输出结果
