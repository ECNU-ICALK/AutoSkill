# 自动化思维与核心信念协同矫正微干预协议

一套标准化的四步协同微干预流程，用于在会谈中即时识别、命名、检验并重构来访者的自动化思维及潜在核心信念，强调证据导向、元认知觉察、行为锚定与现实性校准；整合自动化思维的即时检验与核心信念的矛盾反例探询，聚焦可验证事实、具体情境细节与意义重评。

## Prompt

当来访者表达消极自我陈述或情绪强烈但思维模糊时：① 引导锚定具体自动化思维（例：‘当你感到[情绪]时，那一刻你脑海中闪过的最自然、最直接的想法是什么？请原封不动说出来’），并使用标准术语命名（如‘这是“全或无”思维’）；② 引导寻找支持/反驳该思维的具体事实证据（至少各1条）；③ 若浮现绝对化核心信念（如‘我很无能’‘我永远失败’），启动反例探询：锚定一次与其矛盾的具体胜任事件（‘有没有哪怕一次，你做成了一件事，哪怕很小，当时心里有点踏实或有成就感？’），解构其情境细节（‘它发生在哪里？你做了什么具体动作？什么细节让你觉得‘这事成了’？’），并引导意义重评（‘这件事的发生，和‘我什么都做不好’这个说法，是否存在不一致？’）；④ 协作生成更平衡、可验证的替代思维（避免过度积极，聚焦‘更贴近现实’）。全程嵌入psychoeducation：说明自动化思维的自发性、情绪驱动性，以及核心信念的泛化性与可检视性。

## Triggers

- 来访者出现典型认知扭曲表述
- 情绪骤然下降伴随绝对化语言
- 问题讨论环节中思维僵化明显
- 来访者描述情绪但未关联具体想法
- 来访者用模糊概括性语言表达自我评价
- 情绪强度高但思维内容不清晰
- 来访者使用全称否定词（‘永远’‘全都’‘根本’）表达自我价值
- 信念与具体行为史明显矛盾（如自称‘无能’但曾获学业成就）
- 情绪低落但认知内容可触及
