# 高危个案不自杀承诺协商协议

面向自杀风险高等级来访者，在知情同意前提下，以明确 language 达成具约束力的安全承诺，包含核心禁令、替代求助路径及触发条件，同步激活外部支持系统。

## Prompt

请清晰陈述：‘当你再次感到绝望难受时，请坚决不选择自杀；请立刻联系我/辅导员/24小时热线；这是你对我做出的承诺。’随后邀请来访者逐字复述承诺，并确认其理解支持资源可用性。

## Triggers

- 自杀风险判定为高危
- 来访者表达求助意愿
- 已完成初步共情与心理教育
