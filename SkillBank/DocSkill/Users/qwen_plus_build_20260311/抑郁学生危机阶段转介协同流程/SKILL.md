# 抑郁学生危机阶段转介协同流程

针对大学生抑郁伴高自杀风险个案，整合心理咨询师、辅导员、校医院/精神科三方角色，形成标准化转介响应链：风险确认→知情同意→陪同就医→信息闭环。

## Prompt

当SCL-90等测评显示重度抑郁或临床访谈确认高危时：①向学生解释转介必要性（生理基础+治疗可及性）；②协助预约校医/精神科；③协调辅导员全程陪同；④获得学生授权后向接诊方传递关键风险信息；⑤48小时内跟进诊断结果。

## Triggers

- SCL-90抑郁分≥2.5
- 临床访谈确认高自杀风险
- 学生同意接受医学评估
