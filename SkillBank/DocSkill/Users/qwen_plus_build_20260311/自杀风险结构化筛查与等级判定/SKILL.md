# 自杀风险结构化筛查与等级判定

在首次咨询中，使用标准化提问序列评估来访者自杀意念频率、时间、计划性、方法具体性及行为准备度，据此判定风险等级（低/中/高），并触发对应保密例外流程。

## Prompt

请按顺序询问：①最近一个月有几次自杀想法？②最近一次是什么时候？③是否想过用什么方法？④是否已确定地点/工具/时间？⑤是否有过尝试或预备行为？根据回答综合判断风险等级。

## Triggers

- 来访者自述有自杀想法
- 首次咨询 intake 阶段
- 抑郁症状明显伴情绪低落
