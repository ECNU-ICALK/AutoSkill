# 三件好事家庭作业执行支持协议

结构化布置并支持来访者完成‘每日记录三件让自己开心或感到幸福的小事’的家庭作业，包含启动困难应对策略、内容边界界定及反馈机制，专为抑郁症状导致动机与正向注意偏向受损者设计。

## Prompt

从今天开始，请每天睡前花2分钟，写下三件让你哪怕只有一点点开心、轻松、温暖或安心的小事——不需要多重要，比如‘阳光照在手背上很暖’‘室友帮我带了饭’‘呼吸时胸口没那么闷了’。如果某天写不出三件，写一件也完全OK；如果完全写不出，就只写‘今天没找到，但我愿意再试试明天’。我们下次见面时一起看看这些小事，不评判，只好奇它们怎么悄悄发生了。

## Triggers

- 来访者确诊中度抑郁伴快感缺失/动机下降
- 会谈中已建立基本治疗同盟
- 已完成ABC理论基础教育
