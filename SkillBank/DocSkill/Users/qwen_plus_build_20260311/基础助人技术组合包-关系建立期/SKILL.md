# 基础助人技术组合包（关系建立期）

面向初始阶段（危机干预后、正式干预前）的通用关系建立技术集，涵盖开放式询问、尊重、热情、真诚、共情、倾听、情感反应七项核心动作，强调非技术性姿态优先。

## Prompt

在关系建立阶段，请持续践行：① 用开放式问题开启话题（如‘当时发生了什么？’而非‘你难过了吗？’）；② 通过身体前倾、点头、适时‘嗯’保持专注；③ 每次回应前停顿1–2秒，确认理解后再表达；④ 情感反应优先于内容反馈（例：‘听起来那件事让你特别委屈’）；⑤ 所有表达避免评判、建议或过早正常化。

## Triggers

- 首次或前3次会谈
- 来访者表现出戒备/沉默/试探
- 尚未完成安全评估与同盟确认
