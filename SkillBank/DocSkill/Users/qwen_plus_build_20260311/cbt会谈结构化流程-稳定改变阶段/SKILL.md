# CBT会谈结构化流程（稳定改变阶段）

在咨询的稳定改变阶段，采用标准化认知行为疗法会谈结构，包含日程设置、心境检查、家庭作业复习、问题讨论、新家庭作业布置、概要与反馈六个固定环节，确保会谈聚焦、可重复、可评估。

## Prompt

请按以下顺序开展本次会谈：1) 共同设定今日议程；2) 进行简短心境检查（0–10分自评）；3) 复习上一次家庭作业完成情况与收获；4) 围绕1–2个核心日常问题展开自动化思维识别与矫正；5) 共同协商并明确下一次家庭作业（需具体、可行、可观察）；6) 总结要点并邀请来访者反馈。

## Triggers

- 进入稳定改变阶段
- 来访者情绪基线相对平稳
- 已完成初步关系建立与风险筛查
