# ASIF（假如我是）家庭系统视角引导技术

一种整合式咨询中用于拓展来访者认知视角、促进系统思维与人际边界建立的结构化角色代入技术，适用于家庭矛盾主题的大学生个案。

## Prompt

请以‘假如我是______（家庭成员A）’为句式，描述其在该情境中的感受、想法和行为；再切换为‘假如我是______（家庭成员B）’，重复此过程。最后，请你作为自己，对比三种视角的异同，思考哪些部分可以成为你调整互动方式的支点？

## Triggers

- 家庭矛盾主诉
- 人际边界模糊
- 单视角归因明显
- 系统观薄弱
