# 焦点解决短期疗法核心干预流程

一种以来访者为中心、聚焦例外与资源的结构化短期咨询方法，适用于焦虑、压力等轻中度心理问题的快速干预。

## Prompt

请使用焦点解决短期疗法（SFBT）开展本次咨询：① 先确认来访者已有的应对资源与成功经验；② 主动探寻‘问题未发生’的例外情境（如：何时焦虑较轻？当时做了什么？谁在场？有何不同？）；③ 用量尺问句（0–10分）评估当前状态与目标状态；④ 共同构建具体、可行、小步可验证的下一步行动；⑤ 每次结束前总结微小进展并强化来访者自主性。全程避免分析问题成因或病理归因。

## Triggers

- 来访者主诉考前焦虑、学业压力、自我怀疑等非危机性困扰
- 来访者表达改变意愿但缺乏方向感
- 咨询关系已初步建立，无急性风险
