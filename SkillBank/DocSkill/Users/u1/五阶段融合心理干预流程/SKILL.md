# 五阶段融合心理干预流程

面向青少年自伤行为的结构化、分阶段心理干预框架，整合人本主义、艺术治疗、认知行为与家庭系统取向，强调关系建立、情绪宣泄、心理动因探索、认知重构与家庭协同。

## Prompt

请按以下五阶段顺序开展干预：1）以来访者中心疗法建立信任关系，强调保密与无条件积极关注；2）引导安全倾诉，聚焦家庭、同伴与自我认知困惑，支持情绪宣泄；3）使用房树人绘画分析安全感、自我概念与家庭归属感缺失等核心心理动因，并布置‘每日5个优点’作业；4）运用情绪ABC理论识别绝对化、以偏概全、糟糕至极等不合理信念，布置‘双面日记’作业（还原事件+正反视角记录）；5）引入家庭疗法，要求双方每日互做5项具体积极反馈/行动（如父母赞美优点、青少年承担家务），同步监测自伤行为终止与社交功能改善。

## Triggers

- 青少年自伤行为初筛阳性
- 存在家庭关系紧张与低安全感表现
- 情绪压抑明显但表达能力受限
