# 结构化认知行为会谈流程

在心理咨询稳定改变阶段，采用标准化CBT会谈结构开展单次咨询，确保技术实施的系统性与可重复性。

## Prompt

请按以下顺序执行本次咨询会谈：1）日程设置（与来访者共同确认本次会谈重点）；2）心境检查（使用0–10量尺评估当前情绪状态）；3）家庭作业回顾（简要反馈上期任务完成情况及收获）；4）日常问题讨论（聚焦1–2个具体情境，识别自动化思维）；5）布置新家庭作业（明确、可行、具象化的小步行动或思维记录任务）；6）概要与反馈（总结关键点，邀请来访者复述或补充）。

## Triggers

- 进入稳定改变阶段
- 来访者情绪基线相对平稳
- 已完成初步关系建立与风险筛查
