# 自杀风险青少年的三阶关系-目标-行动咨询工作流

针对有自杀未遂史、低自我认同与高羞耻感的青少年来访者，结构化实施‘关系锚定→目标转化→效能强化’三阶段干预流程，优先修复希望感缺失与自我否定，以可验证家庭作业为行为锚点，推动现实功能改善与自我效能重建。

## Prompt

当面对有自杀未遂史、因体形困扰产生强烈无价值感和社交回避的青少年来访者时，请按以下三阶段执行：1）在首次会谈中，主动识别并镜映其残留的积极资源（如过往学业能力、对父母的情感联结），用具体化肯定替代泛化表扬；2）第二阶段聚焦‘可操作微目标’：协同制定三项家庭作业——健康减重目标（非体重数字，而是行为指标如每日步行30分钟）、期末复习计划（分科分日、含休息时段）、与父母40–60分钟非问题导向的随性对话；3）第三阶段通过行为实验强化效能感：引入榜样参照（相似背景成功案例）、即时正向强化（达标即兑现小奖励），并在系统脱敏中嵌入其真实自豪记忆（如小学演讲）作为催眠暗示锚点。全程保持对自杀意念的动态安全核查。

## Triggers

- 来访者有自杀未遂史
- 自述‘活着没意义’或‘生无可恋’
- 持续贬低自我且否认自身优点
- 将单一外貌问题等同于整体人生失败
