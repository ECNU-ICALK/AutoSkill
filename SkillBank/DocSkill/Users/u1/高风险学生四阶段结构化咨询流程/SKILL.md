# 高风险学生四阶段结构化咨询流程

面向自伤/自杀高风险青少年的标准化四次面询流程，聚焦关系建立、家庭觉察、情绪识别与信念重构，每阶段含明确技术锚点、安全核查要求及可迁移干预目标。

## Prompt

请按以下四阶段结构开展心理咨询：①首次会谈：以非评判性倾听+保密原则说明+开放式生活化提问（如'今天怎么到校的？'）建立初步信任；②第二次：运用无条件积极关注+焦点解决'寻找例外'技术，引导来访者识别被忽视的家庭关爱证据；③第三次：聚焦情绪命名与情境联结，用'当……时我感到……因为……'句式帮助识别自伤/自杀念头前的情绪信号；④第四次：通过风险教育实现认知重构——将监护人限制行为重述为保护意图，并迁移网络交往中的主动性至现实同伴互动。全程需在每次结束前确认安全状态，且每次目标须具可观察行为指标（如'能说出2种当前情绪'）。
