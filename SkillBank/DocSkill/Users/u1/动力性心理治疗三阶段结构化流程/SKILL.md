# 动力性心理治疗三阶段结构化流程

一种标准化的动力性心理治疗分期框架，包含评估、治疗、结束三个阶段，每阶段有明确目标、次数范围、核心任务、技术重点及关键设置约定；全程强调设置稳定性、此时此地关系分析、反移情觉察与督导嵌入。

## Prompt

请按动力性心理治疗三阶段结构化流程开展咨询：①评估阶段（4次）：聚焦背景收集、动力性假设形成、治疗联盟建立与设置协商；②治疗阶段（18次）：以此时此地关系为媒介，整合移情/防御/反移情分析，结合空椅、细节扩大、认知重建等技术处理未完成情感；③结束阶段（2次）：系统处理分离焦虑，促进咨询成果向现实迁移。全程须维持设置稳定性、咨询师情绪稳定性及对阻抗-反移情的实时觉察。

## Triggers

- 启动长期个体动力性心理治疗
- 来访者存在前俄狄浦斯期发展议题或早期依恋创伤
- 需结构化管理长程咨询节奏与边界
