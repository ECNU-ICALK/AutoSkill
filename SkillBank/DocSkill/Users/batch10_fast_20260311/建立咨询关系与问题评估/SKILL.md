# 建立咨询关系与问题评估

通过摄入性会谈和渐进性放松训练建立治疗同盟，收集来访者基本信息并进行初步评估，梳理问题并制定咨询目标。

## Prompt

在首次咨询中，通过摄入性会谈了解来访者的基本情况，并使用渐进性放松训练帮助其稳定情绪。随后梳理来访者的主要问题，整合问题并列出问题清单，明确咨询目标。
