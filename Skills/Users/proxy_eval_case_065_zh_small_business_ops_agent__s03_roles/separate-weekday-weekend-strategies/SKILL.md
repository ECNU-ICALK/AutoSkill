# separate-weekday-weekend-strategies

When creating weekly operation plans for businesses with significant weekday-weekend traffic differences, generate distinct strategies for weekdays and weekends covering personnel scheduling, inventory management, promotional activities, and waste control.

## Prompt

# Goal
Generate a weekly operation plan that includes separate, detailed strategies for weekdays and weekends, covering personnel scheduling, inventory management, promotional activities, and waste control to account for traffic pattern differences.

# Constraints & Style
- Must explicitly separate weekday and weekend sections with clear headings
- Weekday strategies should focus on efficiency and cost control
- Weekend strategies should focus on capacity management and customer experience
- Include specific actionable tasks for each day type
- Use tables or bullet points for clarity
- Keep language concise but comprehensive

# Workflow
1. Identify core operational areas (scheduling, inventory, promotions, waste control)
2. For each area, create weekday-specific strategies
3. For each area, create weekend-specific strategies
4. Include transition/overlap activities for Friday (weekend prep) and Monday (weekend recap)
5. Add implementation notes for flexibility and adjustment
