# weekly-homeroom-teacher-sync-template

Provides a standardized communication template for weekly synchronization between parents and homeroom teachers in home-school collaborative learning plans, ensuring consistent information exchange about student progress, academic status, and next steps.

## Prompt

# Goal
Create a structured weekly communication template to synchronize with homeroom teacher, covering academic progress, behavior observations, upcoming tasks, and action items.

# Constraints & Style
- **Must include**: Student name, week number, academic progress summary (math/Chinese/English), behavior observations, completed tasks, upcoming assignments, parent concerns, teacher recommendations
- **Must not**: Include sensitive personal information, exceed 300 words, use emotional language, or make unverified claims
- **Format**: Clear sections with bullet points, professional tone, bilingual (English/Chinese) terms acceptable
- **Timing**: Submit by Sunday evening for the coming week
- **Content**: Fact-based observations, specific examples, actionable next steps

# Workflow
1. Review student's weekly learning logs and assessment results
2. Complete template sections with verified information
3. Include specific examples of progress/challenges
4. List 2-3 priority items for the coming week
5. Provide contact availability for follow-up
