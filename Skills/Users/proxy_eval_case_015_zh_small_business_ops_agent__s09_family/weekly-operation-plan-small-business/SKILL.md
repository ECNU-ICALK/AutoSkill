# weekly-operation-plan-small-business

创建小企业每周运营计划，整合排班管理、库存管理、促销策略和损耗控制，根据工作日与周末客流差异制定差异化执行方案

## Prompt

# Goal
生成小企业（如咖啡店、餐厅、零售店）的每周运营计划，整合排班管理、库存管理、促销策略和损耗控制流程，根据工作日与周末客流差异制定差异化执行方案。

# Constraints & Style
1. 排班管理：根据历史客流数据设计固定班次（早班/晚班/高峰班），考虑员工技能匹配与排班灵活性
2. 备货管理：建立日常备货流程与周度补货计划，考虑安全库存系数与临期物料处理
3. 促销策略：设计固定促销活动（如会员日）与灵活促销手段（如滞销品清仓），结合社交媒体联动
4. 损耗控制：制定日常损耗管理标准（配方标准化、按需制作），建立周度损耗分析机制
5. 差异化策略：明确工作日（周一-周五）与周末（周六-周日）的客流特征、资源配置、执行重点差异
6. 执行工具：推荐数字化管理工具（如排班APP、库存表格、会员系统）辅助落地
7. 保持计划可操作性，避免过高成本投入，聚焦小而精的服务特色

# Workflow
1. 分析工作日与周末客流特征（高峰时段、客流量、消费行为）
2. 设计分时段排班表（考虑固定班次、高峰班次、灵活替补）
3. 制定基于历史销量的库存采购与日常备货计划
4. 策划分日常与周末的促销活动组合
5. 建立全流程损耗控制标准与分析机制
6. 推荐配套数字化工具并给出实施提示
