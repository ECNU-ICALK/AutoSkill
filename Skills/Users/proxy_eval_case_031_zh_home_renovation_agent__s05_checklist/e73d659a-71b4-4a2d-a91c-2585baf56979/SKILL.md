# 装修工期动态调度执行清单

为应对主材延期等常见扰动，提供可每日执行、角色分离、零解释的刚性施工调度指令，确保总工期不超期且隐蔽工程零妥协。

## Prompt

# Goal
生成一份面向施工队的、可逐日打印执行的刚性作业清单，含日期、工序区域、具体任务、白话验收标准、签字栏；同步生成一份面向业主的极简验收对照清单，仅含3个可快速判断的问题、合格答案和不合格立即动作。

# Constraints & Style
- 严格区分两份输出：施工队版（给工长/班组长）与业主版（给非专业人士），不混用术语；
- 施工队版：每行=1个日期+1个工序区域+1个具体任务+1条白话验收标准（禁用专业缩写，如‘PPR’→‘热水管’，‘ENF’→‘最严环保等级’）；
- 业主版：每个验收环节仅列3个问题，答案必须是可目测/手量/耳听/手机APP验证的二值判断（是/否），不合格动作必须为1句话指令（如‘用红笔圈出，拍照发群：请D10前整改’）；
- 禁止任何解释性文字、背景说明、心法、提醒、模板索取引导；
- 输出即用：施工队版为表格（|日期|工序区域|具体任务|执行标准|验收人签字|），业主版为纯文本分段（无表格、无符号列表）；
- 所有内容必须可单日执行，不含跨日推演或条件分支；
- 不出现品牌名、价格、型号、面积、预算数字等一次性事实；
- 不包含PDF/Word/Excel格式说明或获取引导语。

# Workflow
1. 解析用户指定的延期天数（默认7天）及受影响工序类型（如瓷砖、橱柜、卫浴）；
2. 基于‘隐蔽工程优先、湿作业不交叉、安装类后移’原则，生成D1–D70中关键节点的施工队执行条目（每条对应一个真实可签的当日动作）；
3. 对应生成业主在该节点需完成的极简验收动作（仅3问+判据+拒收指令）；
4. 输出两份独立、去标识、去冗余、可直接打印/截图使用的纯内容。
