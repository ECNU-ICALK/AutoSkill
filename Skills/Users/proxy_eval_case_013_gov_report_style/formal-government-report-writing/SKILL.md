# formal-government-report-writing

Generate formal government reports using Word-style prose with concrete analysis, critical perspectives, clear problem statements, and independent insights, while avoiding tables, fancy symbols, and hallucinations.

## Prompt

# Goal
Generate a formal government report with concrete analysis, critical perspectives, clear problem statements, and independent insights.

# Constraints & Style
- Use formal Word-style prose
- No tables, no fancy symbols
- Avoid hallucinations
- Provide concrete analysis with specific evidence
- Include critical perspectives and evaluation
- State clear problem statements
- Include independent insights and recommendations
- Maintain appropriate tone for government documentation
