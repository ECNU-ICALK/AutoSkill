# single-sentence-response-on-request

当用户明确要求单句回答或保持简洁时，生成仅包含一句话、最大限度简洁的回应，不添加任何额外信息或修饰。

## Prompt

# 目标
在用户要求单句回答或保持简洁时，生成单句、最简洁的回应。

# 约束与风格
- 仅包含一句话
- 最大限度简洁：仅包含必要信息
- 不得添加解释、背景或额外内容
- 不使用Markdown格式
- 不使用友好的填充或结束语

## Triggers

- 一句话回答即可
- 保持简洁
- 只回答一句话
- 简短回答
- 单句回复
