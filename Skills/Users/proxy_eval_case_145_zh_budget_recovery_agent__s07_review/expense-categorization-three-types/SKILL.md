# expense-categorization-three-types

将预算支出明确划分为固定支出、可协商支出和弹性支出三类，作为预算规划的第一步

## Prompt

# Goal
将家庭预算中的支出项目按性质清晰分类为固定支出、可协商支出和弹性支出三类，为后续预算制定提供基础。

# Constraints & Style
- **固定支出**：每月必须支付、金额相对固定的必要开支（如房贷/房租、保险、贷款还款、水电煤气等基本生活开支）
- **可协商支出**：金额可谈判、可调整或可取消的服务费用（如电话费、网络费、订阅服务、会员费）
- **弹性支出**：可根据需要灵活调整的消费（如食品杂货、交通、娱乐、购物、外出就餐等）
- 分类需清晰明确，避免重叠
- 每项支出只能归入一个类别
- 优先识别固定支出和可协商支出，弹性支出为剩余项目

# Workflow
1. 收集所有月度支出项目清单
2. 识别并分类固定支出（必需且不可变更）
3. 识别并分类可协商支出（可通过谈判或取消调整）
4. 将剩余支出归类为弹性支出
5. 核对分类准确性，确保覆盖所有支出项目
