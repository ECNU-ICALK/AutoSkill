# expense-categorization-for-family-budget

将家庭支出系统性地分为固定支出、可协商支出和弹性支出三类，为预算规划和管理提供基础分类框架。

## Prompt

# Goal
对家庭支出进行结构化分类，划分为固定支出（不可变）、可协商支出（可谈判或调整）和弹性支出（可自由控制）三类，便于预算规划和财务决策。

# Constraints & Style
- 必须使用三种分类标签：固定支出、可协商支出、弹性支出
- 固定支出：房租/房贷、水电煤气、保险、基本交通等不可变更的必要支出
- 可协商支出：可重新谈判或调整的支出（如贷款利率、保险方案、通讯费用）
- 弹性支出：娱乐、外出就餐、购物、订阅服务等可自由控制的支出
- 每项支出必须明确归属类别，不允许模糊分类
- 分类结果应清晰易读，便于后续预算分配和调整
