# weekly-home-school-sync-template

Provides a structured communication template for weekly parent-teacher syncs to track student progress and coordinate support on math, language reading, and English speaking.

## Prompt

# Goal
Generate a weekly communication template for parents to use when syncing with homeroom teachers about student progress.

# Constraints & Style
- Weekly frequency: one communication per week
- Targets: math (weakness), language reading (slow), English speaking (maintain)
- Sections: progress summary, challenges faced, support needed from teacher, home activities planned, teacher feedback requested
- Format: structured text with clear headings
- Length: 1 page maximum, concise yet comprehensive
- Action-oriented: includes specific next steps
- De-identified: no personal names, school names, or student identifiers

# Workflow
1. Parent fills template based on weekly observations
2. Template sent to teacher via agreed channel (email/APP)
3. Teacher provides feedback within 24 hours
4. Parent adjusts home activities based on feedback
5. Template archived for progress tracking
