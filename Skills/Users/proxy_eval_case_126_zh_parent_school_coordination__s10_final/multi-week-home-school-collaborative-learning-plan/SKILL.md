# multi-week-home-school-collaborative-learning-plan

创建包含时间分配约束和学科专项训练的家校协同多周学习计划，确保家庭与学校教育活动的结构化配合与进度追踪。

## Prompt

# 目标
生成一个结构化的家校协同多周学习计划，明确每日/每周时间分配，针对不同学科的强弱项设计专项训练活动，并提供家校双方的职责分工与协作机制。

# 约束与风格
- 必须严格遵守指定的时间约束（工作日/周末/每日具体时长）
- 针对数学薄弱、语文阅读速度、英语口语等具体学科问题设计专项训练
- 包含家校双方的明确职责分工与协作活动
- 平衡学术能力、社会情感技能、创造力和身体素质发展
- 提供周/日结构化活动安排与时间分配
- 包含评估与反馈机制（过程性评估、成果性评估、发展性评估）
- 保持灵活性，允许根据学生实际进展调整计划

# 工作流程
1. 识别核心约束条件（时间分配、学科强弱项、适用年级）
2. 确定每周主题焦点（如：第1周建立沟通机制、第2周强化弱项学科、第3周培养社会情感技能、第4周综合展示与评估）
3. 设计每日/每周具体活动安排与时间分配
4. 制定家校双方的职责清单与协作活动
5. 建立评估与反馈机制
6. 提供资源支持与注意事项

# 资源支持
- 家庭学习日志模板
- 学科专项训练指导
- 家校沟通规范
- 评估量表与反馈工具
