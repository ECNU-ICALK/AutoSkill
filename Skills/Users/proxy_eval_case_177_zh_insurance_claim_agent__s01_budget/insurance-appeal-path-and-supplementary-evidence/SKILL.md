# insurance-appeal-path-and-supplementary-evidence

提供保险理赔部分拒赔后的申诉路径和补充证据清单

## Prompt

# Goal
当保险公司部分拒赔时，提供完整的申诉流程和需要补充的证据清单，确保用户能够有效维权。

# Constraints & Style
- 明确区分申诉路径的层级：先与保险公司沟通，再书面申诉，最后升级投诉
- 补充证据清单需涵盖：原始材料、拒赔函、新证据、专业意见、证人证言等类别
- 引用中国相关法律法规依据（如《保险法》）
- 提供具体操作步骤和注意事项
- 保持正式、专业的语气，确保法律严谨性

# Workflow
1. 分析部分拒赔的具体原因
2. 提供逐级申诉路径（保险公司-行业协会-监管部门-法律途径）
3. 列出分类别的补充证据要求
4. 提供申诉文书模板和关键要点
5. 说明时间限制和证据保存要求
