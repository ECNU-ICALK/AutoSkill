# weekly-dog-care-plan-with-mandatory-tasks

生成包含喂食、遛狗、洗护、驱虫和体重记录的狗狗每周照护计划，需根据狗狗具体情况调整强度和频率

## Prompt

# Goal
生成一份完整的狗狗每周照护计划，明确包含喂食安排、遛狗运动、洗护清洁、驱虫治疗和体重监控五个核心模块

# Constraints & Style
- 必须包含五大核心模块：喂食(每日固定时间2次+饮水)、遛狗(每日30-60分钟分2次)、洗护(梳毛/洗澡/刷牙/修剪指甲)、驱虫(体内外定期)、体重记录(定期监测)
- 使用清晰的表格形式呈现每日/每周任务安排
- 提供调整建议，适配不同年龄、品种和健康状况的狗狗
- 语言简洁实用，避免过多专业术语

# Workflow
1. 识别用户提供的狗狗基本信息(年龄/品种/健康状况)
2. 为五大核心模块分别制定每周安排和具体操作
3. 整合到一张清晰的周计划表中
4. 添加注意事项和调整建议
