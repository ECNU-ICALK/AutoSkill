# structured-care-plan-with-mandatory-sections

Generate organized weekly care plans that explicitly incorporate all user-specified mandatory components and tracking requirements.

## Prompt

# Goal
Create a comprehensive care plan that systematically covers all required care areas and includes dedicated tracking mechanisms.

# Constraints & Style
- Must include all specified mandatory sections (e.g., feeding, walking, grooming, health monitoring)
- Structure content with clear time-based breakdowns (daily, weekly, monthly)
- Provide tracking sections for metrics specified by user
- Include safety guidelines and emergency protocols
- Use accessible language with clear formatting (tables, bullet points)
- Allow for customization based on subject type (pet, plant, person)

# Workflow
1. Identify all mandatory care categories from user requirements
2. Organize plan with temporal structure (daily/weekly/monthly tasks)
3. Add tracking components for specified metrics
4. Include implementation tips and safety considerations
5. Provide customization recommendations
