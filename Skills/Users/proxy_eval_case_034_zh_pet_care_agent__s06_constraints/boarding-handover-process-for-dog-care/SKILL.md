# boarding-handover-process-for-dog-care

A skill to create and integrate a boarding handover process into a dog care plan, including pre-handover preparation, information transfer, and post-return protocols.

## Prompt

# Goal
Create a structured boarding handover process for a dog care plan, ensuring continuity of care when transferring responsibility to another caregiver for a specified period.

# Constraints & Style
- Must include pre-handover preparation checklist (food, meds, supplies, emergency contacts)
- Must provide clear care instructions for the caregiver (feeding schedule, walking routine, medical needs)
- Must include post-return review procedures (health check, behavioral assessment)
- Use clear, actionable steps with time-bound milestones
- Include contingency planning for emergencies
- Maintain consistency with existing weekly care plan components

# Workflow
1. Pre-handover phase (3-5 days before departure):
   - Prepare care package with food, meds, cleaning supplies
   - Document current health status and care routines
   - Arrange emergency contacts and vet details
   - Conduct trial session with caregiver

2. Handover execution (departure day):
   - Physical handover with verbal explanation
   - Provide written care instructions
   - Ensure caregiver understands all requirements
   - Confirm communication channels

3. Ongoing monitoring (during boarding period):
   - Regular check-ins (daily or as needed)
   - Maintain care log updates
   - Address issues promptly

4. Return and transition (after boarding):
   - Gradual reintegration period
   - Health and behavioral assessment
   - Feedback collection and plan updates
