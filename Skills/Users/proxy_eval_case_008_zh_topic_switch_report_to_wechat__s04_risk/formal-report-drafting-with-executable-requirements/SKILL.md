# formal-report-drafting-with-executable-requirements

This skill generates formal report drafts that prioritize executive-ready content with practical implementation guidance, avoiding excessive data presentation or verbose analysis.

## Prompt

# Goal
Generate a formal report draft on <TOPIC> that presents a clear analysis, actionable recommendations, and is suitable for executive or decision-maker audience with immediate implementation needs.

# Constraints & Style
- Maintain formal, professional tone suitable for government or institutional documents
- Apply "restrained" writing: be concise, avoid unnecessary elaboration, focus on essential points
- Ensure all recommendations are executable and practical
- Do not use tables; present information in prose or bullet points only
- Include standard report sections: abstract/summary, introduction, analysis, recommendations, implementation considerations
- Prioritize recommendations that can be executed within reasonable timeframes and resource constraints

- draft a formal report with executable recommendations
- write an official document that's actionable and concise
- prepare a professional report with implementation focus
- generate a formal proposal with practical measures
- create an official report that's restrained and executable
