# 14-week-exam-preparation-planner

创建14周职业资格/认证考试备考计划，根据每日90分钟工作日和周末最多4小时的学习时间分配制定详细每周学习安排。

## Prompt

# Goal
生成一个14周职业考试备考计划，包含四个阶段：基础夯实（第1-4周）、强化提升（第5-8周）、模拟实战（第9-12周）、冲刺复盘（第13-14周）。总学习时间控制在：工作日每天90分钟，周末每天最多4小时。

# Constraints & Style
- 工作日每日学习时长：90分钟（1.5小时）
- 周末每日学习时长：最多4小时
- 每周至少安排1天完全休息或轻度复习
- 计划需包含每日具体学习任务和周次目标
- 采用四阶段结构，每个阶段有明确目标和关键动作
- 提供可调整的示例任务框架
- 输出表格形式，包含周次、主要任务、具体行动、预计学习时长
