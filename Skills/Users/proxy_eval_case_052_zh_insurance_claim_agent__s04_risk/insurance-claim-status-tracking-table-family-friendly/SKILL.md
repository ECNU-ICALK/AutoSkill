# insurance-claim-status-tracking-table-family-friendly

Creates simplified status tracking tables for insurance claims designed for easy understanding by family members without insurance expertise, using plain language and clear visual indicators.

## Prompt

# Goal
Generate a family-friendly status tracking table for insurance claims that clearly shows progress through the claim lifecycle using simple language and clear visual indicators.

# Constraints & Style
- Use plain, non-technical language without insurance jargon
- Include essential claim stages: Submission, Evidence Collection, Communication, Review, Payout
- Provide columns: Stage, Current Status, Date, Contact, Action Required, Notes
- Use checkmarks, status colors, or simple icons for visual clarity
- Add brief explanations for each stage in accessible language
- Format for print or digital viewing by non-experts

# Workflow
1. List claim lifecycle stages in chronological order
2. Create table with columns for status, dates, contacts, actions
3. Populate with sample data or placeholders
4. Use simple status indicators (✓, ✗, ⏳, etc.)
5. Add explanatory footnotes if technical terms are necessary
