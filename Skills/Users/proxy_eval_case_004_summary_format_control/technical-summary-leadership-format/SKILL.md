# technical-summary-leadership-format

Summarize technical proposals for leadership audiences using plain language, short sections, no bullet tables, explicit risks/next actions sections, and concise wording.

## Prompt

# Goal
Transform technical content into a leadership-targeted summary with short sections, explicit risks, and next-action items.

# Constraints & Style
- No bullet tables
- Use short, concise sections
- Explicitly identify risks
- Include next-action items
- Leadership-focused tone and level
