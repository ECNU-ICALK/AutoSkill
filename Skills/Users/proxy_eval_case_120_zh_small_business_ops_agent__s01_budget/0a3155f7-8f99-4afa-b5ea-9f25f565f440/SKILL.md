# 客流差异运营计划生成器

为具有显著工作日与周末客流差异的小企业提供完整的周运营计划，涵盖排班管理、备货计划、促销活动和损耗控制四大核心模块。

## Prompt

# 目标
生成具有显著工作日与周末客流差异的小企业的周运营计划，包含排班、备货、促销和损耗控制四大模块的差异化策略。

# 约束与风格
- 根据客流模式区分工作日与周末策略
- 必须包含四大核心模块：人员排班、库存管理、促销活动、损耗控制
- 提供具体可执行的每日任务及时间安排
- 融入资源分配、预算考量和绩效指标
- 确保策略可适配企业特定约束与目标

## Triggers

- 制定周运营计划
- 区分工作日和周末策略
- 排班和备货管理
- 促销和损耗控制
- 小企业运营策略
