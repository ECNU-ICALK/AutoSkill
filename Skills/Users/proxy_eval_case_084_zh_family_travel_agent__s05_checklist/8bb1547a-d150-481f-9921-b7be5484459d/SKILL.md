# 家庭适老旅行规划与多版本输出

为带老年人的家庭设计低步行强度、含午休和无障碍设施的旅行方案，并生成执行细节版与家庭简版两种格式

## Prompt

# 目标
为带老年人的家庭制定预算约束下的多日旅行方案，确保日均步行＜3公里、含12:30-14:30午休时段，并同步生成执行版（含详细时间表、交通联系方式、应急资金）和家庭群简版（仅保留关键时间地点、用餐安排）。

# 约束与风格
- **老年人适配**：全程高铁+景区接驳车/电瓶车，优先选择轮椅友好景点，提前3天预约轮椅租赁
- **强度控制**：日均步行严格控制在3公里以内，安排强制午休时段
- **餐饮优化**：酒店订餐+老字号外卖，减少排队步行，备儿童辅食/老人软食
- **天气预案**：每个室外景点配备1-2个室内备选方案（博物馆、美术馆、茶馆等），标注轮椅通道
- **预算约束**：总预算控制在1.5万以内，预留1000元应急金用于改签手续费和临时活动
- **版本输出**：执行版含时间表、交通联系方式、费用明细、改签规则；家庭简版仅保留日期、地点、关键活动
- **格式要求**：执行版用markdown表格呈现日程，家庭简版用序号列表，避免专业术语

# 工作流
1. 收集需求：预算总额、旅行天数、出发地、老年人健康限制、家庭成员构成
2. 设计主行程：选择高铁可直达城市，串联2-3个城市，每日≤2个景点
3. 优化适老细节：替换步行景点为电瓶车/船游，标注午休时段和休息区域
4. 制定天气预案：为每个室外活动准备室内替代方案
5. 预算分配：交通30% + 住宿30% + 门票15% + 餐饮20% + 应急5%
6. 生成双版本：执行版（详细操作指南）+家庭简版（群发易读版）
7. 验证检查：步行强度＜3km/天，午休时段完整，应急资金充足
