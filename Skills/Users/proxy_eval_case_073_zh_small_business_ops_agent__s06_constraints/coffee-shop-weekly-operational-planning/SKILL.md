# coffee-shop-weekly-operational-planning

为小型咖啡馆制定包含排班、备货、促销、损耗控制的每周运营计划，区分工作日与周末客流差异策略

## Prompt

# 目标
制定咖啡小店每周运营计划方案，包含排班、备货、促销、损耗控制四个核心环节，明确区分工作日与周末策略差异。

# 约束与风格
- 必须包含排班策略、备货管理、促销活动、损耗控制四大模块
- 工作日与周末采用差异化策略：根据客流高峰时段、人员配置、备货量、促销力度等维度设计
- 排班：提供基础班次表（早班/晚班/午休轮换）和灵活调整机制（天气/活动预测、实时调度）
- 备货：给出备货公式（日均销量×预估天数×安全库存系数）、库存监控表模板、供应商合作技巧
- 促销：设计固定活动（如周四会员日）、联名活动、社交媒体推广方案
- 损耗控制：从原料、设备、人为三方面制定具体措施和奖惩机制
- 语言简洁实用，提供可直接落地的表格模板和执行工具推荐
- 使用咖啡店常见物料名称（咖啡豆、牛奶、糖浆等）作为示例，但不绑定具体数值
- 不添加未提及的通用标准或流程

# 约束
- 工作日（周一-周五）与周末（周六-周日）策略必须分开设计
- 每个模块需包含可执行的操作步骤和量化指标
- 提供Excel/表格模板示例，标注关键填写项
- 语言风格：专业实用，适合咖啡店管理者直接使用
