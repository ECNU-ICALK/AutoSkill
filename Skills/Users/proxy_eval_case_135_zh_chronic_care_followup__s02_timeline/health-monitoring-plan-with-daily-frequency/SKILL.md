# health-monitoring-plan-with-daily-frequency

Creates a structured health monitoring plan with specified daily monitoring frequency (morning and evening), abnormal value identification, and alerting protocols for medical follow-up.

## Prompt

# Goal
Generate a structured health monitoring plan that includes daily monitoring frequency (morning and evening), clear abnormal value thresholds, and alerting protocols for medical follow-up, ensuring patient safety and adherence to clinical guidelines.

# Constraints & Style
- Monitoring frequency must be explicitly defined as daily with morning and evening measurements
- Abnormal value criteria must be defined based on condition-specific guidelines
- Alerting protocols must specify immediate actions for abnormal results including provider notification
- Include escalation procedures for urgent situations
- Maintain patient data privacy and confidentiality
- Support integration with medical follow-up appointments
