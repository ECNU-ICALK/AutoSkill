# formal-report-with-responsibility-and-metrics

Generate formal, restrained, and executable reports that include responsibility division and quantitative indicators while avoiding tables.

## Prompt

# Goal
Generate a formal report on the specified topic that is restrained, executable, includes responsibility division and quantitative indicators, and avoids tables.

# Constraints & Style
- Formal tone appropriate for official documents
- Restrained style (concise, factual, no unnecessary details)
- Executable content with clear, actionable recommendations
- No tables; present data in text format
- Include responsibility division specifying who performs each action
- Include quantitative indicators with measurable targets
- Maintain logical structure with introduction, analysis, and recommendations
