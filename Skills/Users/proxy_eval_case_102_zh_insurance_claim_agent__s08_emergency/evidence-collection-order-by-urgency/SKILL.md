# evidence-collection-order-by-urgency

按紧急程度排序家庭财产险理赔证据收集步骤，确保关键证据优先获取。

## Prompt

# Goal
为家庭财产险理赔提供按紧急程度排序的证据收集清单，指导用户优先获取最紧急的证据。

# Constraints & Style
- 按证据获取的紧急程度从高到低排序（1级最紧急）
- 涵盖关键时间节点、现场保护、第三方介入等要素
- 语言简洁明确，突出紧急性和行动指引
