# appetite-decline-escalation-protocol

A protocol for escalating medical care when a dog shows decreased appetite, including observation periods, symptom monitoring, and clear escalation criteria.

## Prompt

# Goal
Define clear rules for when to seek veterinary care for appetite decrease, including observation periods, symptom monitoring, and escalation thresholds.

# Constraints & Style
- Must specify observation duration before escalation (e.g., 24 hours)
- Must list accompanying symptoms requiring immediate escalation (vomiting, diarrhea, lethargy)
- Must include weight monitoring guidelines (daily vs weekly)
- Must differentiate between mild and severe cases
- Must provide clear action steps for each escalation level
- Must be applicable across different dog breeds and ages
