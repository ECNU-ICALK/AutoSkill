# medical-monitoring-reminder-system

设计医疗监测随访提醒系统，基于每日双时段测量和异常指标触发非侵入式简洁提醒机制。适用于慢性病（如高血压）的日常管理场景。

## Prompt

# Goal
构建医疗监测提醒系统，实现每日早/晚双时段测量记录，根据预设异常阈值触发适当频率的简洁随访提醒，避免过度打扰患者/家属。

# Constraints & Style
- 必须包含每日早/晚双时段测量时间点提醒
- 异常触发条件需明确量化（如血压值、持续天数、伴随症状）
- 提醒频率需控制在非侵入式水平（每日不超过3次基础提醒）
- 提醒内容必须简洁（核心数据+关键动作，不超过3行）
- 需提供手动监督机制作为自动化补充
- 必须包含紧急情况即刻就医触发规则
- 避免使用手腕式血压计，推荐上臂式设备

# Workflow
1. 设定基础测量提醒：每日早晨起床后1小时、睡前1小时
2. 设定异常监测规则：
   - 持续3天指标超标
   - 单次指标达急症阈值
   - 伴随特定症状
3. 配置提醒通道：手机日历/闹钟 + 智能设备推送（可选）
4. 设定监督机制：家属每周检查记录
5. 建立紧急情况直连机制

## Triggers

- 慢性病日常监测提醒
- 异常指标触发随访
- 非侵入式提醒设计
- 双时段测量记录
- 紧急情况即刻响应
