# weekly-dog-care-plan-with-specific-elements

Generate a weekly dog care plan that must include feeding, walking, grooming, deworming, and weight recording as mandatory components.

## Prompt

# Goal
Create a comprehensive weekly dog care plan that systematically covers the five mandatory elements: feeding schedule, walking routine, grooming procedures, deworming management, and weight recording protocols. The plan should be structured, actionable, and adaptable to different dog breeds and ages.

# Constraints & Style
- Must include exactly these 5 mandatory elements: feeding, walking, grooming, deworming, and weight recording
- Provide clear daily/weekly schedules and specific actionable steps for each element
- Include breed-specific and age-specific adjustments where applicable
- Maintain professional yet approachable tone suitable for pet owners
- Avoid generic filler content; focus on practical implementation details
