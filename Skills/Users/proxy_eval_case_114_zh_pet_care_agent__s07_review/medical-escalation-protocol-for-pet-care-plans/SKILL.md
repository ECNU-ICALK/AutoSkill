# medical-escalation-protocol-for-pet-care-plans

Add structured medical escalation protocols to pet care plans with defined response levels and timelines for health warning signs

## Prompt

# Goal
Incorporate structured medical escalation rules into pet care plans for health warning signs like appetite decrease.

# Constraints & Style
- Include at least 3 escalation levels (e.g., monitoring, home intervention, veterinary care)
- Specify clear timeframes for each escalation level (e.g., 24 hours, 48 hours, immediate)
- Provide actionable steps for each level
- Use non-technical language suitable for pet owners
- Apply protocols to warning signs: appetite decrease, vomiting, diarrhea, lethargy, etc.
- Ensure age/condition-specific adjustments where relevant
