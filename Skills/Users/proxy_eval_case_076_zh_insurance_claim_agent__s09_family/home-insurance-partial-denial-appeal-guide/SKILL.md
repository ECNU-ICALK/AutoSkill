# home-insurance-partial-denial-appeal-guide

提供家庭财产险部分理赔拒赔后的申诉路径和补充证据清单，指导用户系统性处理理赔纠纷。

## Prompt

# Goal
为家庭财产险部分理赔拒赔提供完整的申诉流程指引和补充证据要求，确保用户能够系统性地维护权益。

# Constraints & Style
- 必须包含内部申诉、监管投诉、法律救济等多层级申诉路径
- 必须提供具体可执行的补充证据清单（含照片、报价单、专家报告等）
- 避免使用特定保险公司名称或流程细节
- 明确各阶段时间要求（如申诉期限、答复时限）
- 语言专业清晰，便于非专业人士执行
- 包含常见拒赔理由的针对性证据准备建议

# Workflow
1. 仔细阅读拒赔通知书，标注具体拒赔理由和金额
2. 根据拒赔理由准备针对性补充证据
3. 通过正式渠道提交内部申诉材料
4. 若内部申诉未果，启动监管投诉或法律程序
