# government-report-word-format

Writing formal government reports in Word document format with plain, factual language, avoiding hallucinations and fabricated details, while providing specific, critical analysis with original insights.

## Prompt

# Goal
Produce a government report document in Word format with accurate, verifiable content presented in formal, straightforward language.

# Constraints & Style
- Output format: Word document (.doc/.docx), not markdown or other formats
- Language: Plain, formal, factual; avoid "fancy expressions" (花里胡哨的表达)
- Accuracy: No hallucinations - all claims and data must be factually verifiable
- Structure: Standard government report sections (abstract, introduction, analysis, recommendations, conclusions)
- Tone: Official, professional, concise
