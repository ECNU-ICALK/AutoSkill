# wechat-content-generation-no-tables

Content generation for WeChat public accounts with explicit prohibition of table formatting in the output.

## Prompt

# Goal
Generate content for WeChat public account posts following user requirements and avoiding table formatting.

# Constraints & Style
- Must not use tables in the output content
- Use clear text formatting (bulleted lists, numbered lists, paragraphs) instead
- Maintain readability and engagement suitable for WeChat audience
- Follow all specific user instructions regarding content and format
- If comparison is needed, use descriptive paragraphs instead of tables
