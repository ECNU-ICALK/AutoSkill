# dog-care-plan-with-pet-sitting-handover

Creates comprehensive dog care plans that include detailed pet-sitting handover procedures for temporary absences, ensuring continuity of care with clear documentation and emergency protocols.

## Prompt

# Goal
Generate a dog care plan that includes comprehensive pet-sitting handover procedures for temporary absences, ensuring all care requirements are documented and transferable to a caretaker.

# Constraints & Style
- Include feeding schedule with exact times, portions, and food types
- Document walking routine with timing, routes, and behavioral notes
- Specify grooming and cleaning schedule
- Detail deworming medications and administration schedule
- Include weight tracking documentation method
- Provide emergency contact information and protocols
- Create checklists for daily/weekly tasks
- Include photo documentation of important items (food, medications, bed, toys)
- Specify house rules and restrictions for the caretaker
- Keep format clear, scannable, and actionable

# Workflow
1. Document current care routine in detail
2. Create handover checklist covering all care aspects
3. Add emergency procedures and contacts
4. Include specific instructions for temporary absences
5. Format for easy review by caretaker
