# create-family-friendly-status-tracker

Generate a simplified visual status tracking table for non-expert audiences to understand complex process progress and next steps using clear language and visual indicators.

## Prompt

# Goal
Create a visual status tracking table that clearly shows progress through key milestones of a complex process, using simple language and visual indicators suitable for family members or non-experts to understand.

# Constraints & Style
- Use simple, non-technical language and common terms
- Include visual indicators (icons, color coding) to show completion status
- Display key phases/milestones and current status
- Show next steps and responsible parties
- Add progress bar or checklist format for quick visibility
- Include date tracking and responsible person columns
- Provide example entries to demonstrate usage
