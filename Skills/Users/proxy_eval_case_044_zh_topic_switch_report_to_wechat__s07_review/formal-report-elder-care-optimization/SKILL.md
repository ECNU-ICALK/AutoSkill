# formal-report-elder-care-optimization

生成社区养老服务优化主题的正式报告草案，要求采用正式克制文风、包含可执行建议、不使用表格、明确责任分工并设置量化指标。

## Prompt

Follow this SOP (replace specifics with placeholders like <PROJECT>/<ENV>/<VERSION>):
1) *关于社区养老服务优化的正式报告草案**
2) *标题：**
3) 社区养老服务优化方案建议报告
4) *摘要：**
5) 本报告旨在分析当前社区养老服务的现状与挑战，提出系统性优化建议，以提升服务质量和老年人生活质量。通过整合政策支持、资源整合与技术创新，推动社区养老服务向专业化、智能化、人性化方向发展。
6) *一、引言**
7) 随着我国人口老龄化趋势加剧，社区养老服务作为养老体系的重要组成部分，其优化升级已成为社会关注的焦点。本报告基于对现有服务模式的调研，结合国内外先进经验，提出针对性改进措施，助力构建可持续、高质量的社区养老服务体系。
8) *二、现状分析**
9) **服务覆盖不足**
10) 部分社区养老服务设施陈旧，服务项目单一，难以满足多样化需求。

For each step, include: action, checks, and failure rollback/fallback plan.
Output format: for each step number, provide status/result and what to do next.
