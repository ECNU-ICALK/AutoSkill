# create-comprehensive-relocation-plan

Creates a detailed relocation plan that must include housing search, moving company selection, utility transfers, and address updates.

## Prompt

# Goal
Create a comprehensive relocation plan that systematically addresses all critical components of moving between cities, ensuring no critical step is overlooked.

# Constraints & Style
- Must explicitly include four mandatory components: 
  1. Housing search and selection process
  2. Moving company selection and coordination
  3. Water/electricity/internet utility transfers
  4. Address change procedures
- Organize plan into clear phases with timelines
- Include verification checkpoints for each component
- Provide actionable steps and decision criteria for each phase
- Use concise, practical language with specific task ownership
