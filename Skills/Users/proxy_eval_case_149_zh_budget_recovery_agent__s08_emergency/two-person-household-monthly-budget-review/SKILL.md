# two-person-household-monthly-budget-review

为双人家庭设计月度预算复盘流程，包括支出分析、偏差诊断、计划调整和执行评估的完整工作流程。

## Prompt

# Goal
执行双人家庭月度预算复盘，分析本月支出情况，识别偏差原因，制定下月优化计划，确保预算体系持续改进。

# Constraints & Style
1. 针对双人家庭（2名成员）设计，确保流程简洁高效
2. 必须包含：支出统计、类别分析、偏差诊断、计划调整、执行评估五大环节
3. 输出需包含可执行的下月优化措施，而非仅总结
4. 考虑双人家庭的协作决策机制，明确审批流程
5. 强调从历史数据中学习和持续改进的原则

# Workflow
1. 数据收集阶段（月初1-2日）
   - 汇总本月所有支出记录，按固定/可协商/弹性三类分类统计
   - 计算各类别预算执行率（实际支出/预算总额）
   - 统计应急支出使用情况及审批记录

2. 偏差分析阶段（月初3-4日）
   - 识别超支类别（执行率>105%）和超支金额
   - 分析超支根本原因（计划偏差/意外支出/执行不力）
   - 评估纠偏机制触发情况及效果

3. 决策会议阶段（月初5日）
   - 召开双人家庭会议，展示分析结果
   - 共同讨论偏差原因及改进方案
   - 审批下月预算调整方案

4. 计划制定阶段（月初6-7日）
   - 调整各类别预算分配（超支类别下调15-20%）
   - 制定针对性控制措施（如冻结机制、审批升级）
   - 更新应急基金目标余额

5. 执行监控阶段（当月剩余时间）
   - 每日记录支出，每周复盘执行情况
   - 连续两周超支自动触发纠偏机制
   - 建立透明沟通机制，确保双方知情权
