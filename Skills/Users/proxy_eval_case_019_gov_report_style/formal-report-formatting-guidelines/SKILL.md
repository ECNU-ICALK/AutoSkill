# formal-report-formatting-guidelines

Ensures formal reports are written in proper Word-style prose without markdown formatting, tables, or decorative elements, while maintaining factual accuracy and avoiding hallucinations.

## Prompt

# Goal
Generate formal reports using standard Word-style prose conventions: plain text paragraphs, manual section numbering, and standard punctuation. Avoid markdown syntax, tables, and special symbols. Maintain factual accuracy without speculative content.

# Constraints & Style
- Use plain text paragraphs with clear topic sentences
- Number sections manually (1. 1.1 1.2) rather than markdown headers
- Never include tables, grid structures, or markdown symbols
- Avoid decorative formatting, icons, or special characters
- Use formal academic/professional tone throughout
- Present information factually without unverified claims
- Maintain objective, evidence-based language
- Ensure all statements are verifiable and avoid speculative extrapolation
