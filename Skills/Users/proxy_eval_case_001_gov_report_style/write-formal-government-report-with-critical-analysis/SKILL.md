# write-formal-government-report-with-critical-analysis

Generates formal government-style reports that provide concrete, critical analysis with clear problem statements and independent insights, while maintaining strict factual accuracy and adhering to formal Word-style prose conventions.

## Prompt

# Goal
Transform technical or policy subject matter into formal government reports that offer critical analysis, identify concrete problems, and provide independent insights while maintaining strict factual accuracy and professional writing standards.

# Constraints & Style
- Use formal Word-style prose structure (executive summary, introduction, findings, analysis, recommendations, conclusion sections)
- Avoid tables, tabular data presentations, and non-standard formatting
- Eliminate decorative symbols, emojis, and non-standard formatting elements
- Maintain strict factual accuracy - avoid hallucinations, speculative content, or unverified claims
- Provide concrete, critical analysis with explicit problem statements
- Include independent insights that go beyond surface-level description or summary
- Use professional tone appropriate for official government publications
- Balance depth with conciseness while ensuring completeness of analysis

# Workflow
1. Identify and structure key sections required for government report format
2. Develop clear problem statements for the subject matter
3. Conduct critical analysis of implications and challenges
4. Formulate independent insights and recommendations
5. Verify all claims against established evidence and research
6. Format output using formal Word-style prose conventions without tables or decorative elements
