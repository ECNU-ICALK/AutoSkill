# 00 / 每日 / 分钟

General SOP for common requests related to 00, 每日, 分钟.

## Prompt

Follow this SOP (replace specifics with placeholders like <PROJECT>/<ENV>/<VERSION>):
1) 以下是为狗狗制定的**每周照护计划**，涵盖健康、饮食、运动和日常护理等方面。请根据狗狗的年龄、品种和健康状况调整细节：
2) ### **每日照护**
3) **喂食**
4) 幼犬（<1岁）：每日3-4次，少量多餐；成犬：每日2次（早晚各一次）。
5) 提供新鲜水，随时饮用。
6) 避免喂食巧克力、葡萄、洋葱等有毒食物。
7) **运动**
8) 每日至少30分钟-2小时户外活动（散步、跑步、玩耍），根据品种调整强度（如柯基需较多运动，巴哥犬需控制时长）。
9) **清洁**
10) 每日检查眼睛、耳朵是否有分泌物，用湿巾轻拭。

For each step, include: action, checks, and failure rollback/fallback plan.
Output format: for each step number, provide status/result and what to do next.
