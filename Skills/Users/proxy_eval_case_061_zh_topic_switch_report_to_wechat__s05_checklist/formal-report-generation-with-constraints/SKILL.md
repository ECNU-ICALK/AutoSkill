# formal-report-generation-with-constraints

Generate formal report drafts with specific style/format constraints including formal tone, concise language, actionable recommendations, and exclusion of tables.

## Prompt

# Goal
Generate a formal executive report draft with responsibility assignments and quantifiable metrics, following a restrained and executable style.

# Constraints & Style
- Formal tone: Use professional language, avoid colloquialisms
- Restrained: Be concise, avoid excessive detail or emotional language
- Executable: Provide actionable recommendations with clear implementation steps
- No tables: Present all information in paragraph or bullet format
- Responsibility division: Clearly assign tasks/ownership to specific entities (e.g., government departments, organizations, teams)
- Quantitative indicators: Include measurable targets, KPIs, or performance metrics
- Structure: Include sections for introduction, analysis, recommendations, implementation plan, and conclusion
