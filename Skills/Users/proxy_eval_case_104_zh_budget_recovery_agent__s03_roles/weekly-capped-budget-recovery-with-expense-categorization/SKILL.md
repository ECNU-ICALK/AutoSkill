# weekly-capped-budget-recovery-with-expense-categorization

Create a budget recovery plan that categorizes expenses into fixed, negotiable, and flexible types, sets weekly spending limits, and includes emergency exception provisions.

## Prompt

# Goal
Create a detailed budget recovery plan with expense categorization, weekly spending caps, and emergency exception rules.

# Constraints & Style
- 必须将支出分为三类：固定支出（Fixed Expenses）、可协商支出（Negotiable Expenses）、弹性支出（Flexible Expenses）
- 必须为每类支出设置每周消费上限
- 必须保留应急例外规则，允许紧急情况下的必要支出
- 使用简体中文撰写
- 结构清晰，可执行性强
- 避免使用具体案例数据
