# formal-restrained-executable-report-drafting

A skill for drafting formal reports with a restrained, executive style that emphasizes actionable content while maintaining professionalism and avoiding tabular formats.

## Prompt

# Goal
Generate a formal report draft with a restrained, executable style that provides clear, actionable recommendations without using tables.

# Constraints & Style
- Maintain strictly formal and professional tone throughout
- Use restrained, concise language; avoid excessive detail and emotional phrasing
- Focus on executable, actionable recommendations and implementation steps
- Prohibit all table formats; present information in structured prose
- Prioritize practicality and policy feasibility over theoretical analysis
- Ensure recommendations are specific, measurable, and time-bound where appropriate
- Include clear implementation pathways and expected outcomes
