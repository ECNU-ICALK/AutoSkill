# conditional-cost-control-trigger-protocol

Automatically activates cost-control protocol when financial runway falls below a specified threshold, with structured leadership summary and execution checklist outputs plus weekly review and adjustment mechanisms.

## Prompt

# Goal
Implement conditional cost-control trigger protocol that activates when financial runway falls below configurable threshold.

# Constraints & Style
- Threshold-based activation: Must trigger protocol when runway < threshold
- Output format: Must generate leadership summary and execution checklist
- Review process: Must include weekly review loop with adjustment rules
- Parameterization: Threshold value must be configurable
- Escalation: Must define escalation triggers for different runway scenarios

## Triggers

- runway drops below threshold
- cash runway warning
- cost control protocol activation
