# 国自然基金标题精炼与协同强化

为国家自然科学基金申请生成符合评审要求的正式、简洁、机理清晰的项目标题，严格规避政策性词汇与空泛修饰，聚焦科学问题属性、研究对象、机制/关系/原理等核心要素，并在含多个核心概念时显式嵌入可验证的逻辑关系动词，确保语法与语义层面形成不可分割的科学整体。

## Prompt

# Goal
生成适用于国家自然科学基金（NSFC）申请的正式、精炼、学术严谨的项目标题，长度控制在22–26字以内（含中文标点），突出基础性、机理性与学科交叉特征，满足形式审查与专家初筛双重要求；当标题含两个及以上核心科学要素（如'语义认知'与'时间演化'）时，须显式建立因果、约束、耦合、驱动、交互、演化依赖等可检验关系，消除并列割裂感。

# Constraints & Style
- 必须使用规范学术术语：'时间演化'（不用'时序演化'）、'语义认知'（不用'语义理解'或简写为'语义'）、'决策智能体'（不简写为'智能体'或'AI agent'）；
- 禁用所有非学术修饰词：'新型''前沿''赋能''使能''范式''新一代''智能''自主''浅析''初探''关于''基于''面向'（除非语法必需且无歧义）；
- 禁用模糊动词：'融合''构建''研究'（仅可在副标题/冒号后谨慎使用）；优先采用'机制''关系''耦合''约束''驱动''表征''建模''交互''适应''原理''稳定性''演化性'等NSFC高频可证伪、可分解的科学动词；
- 若含两个及以上核心要素，必须使用强逻辑动词短语显式连接（如'受…约束''由…驱动''在…耦合作用下''基于…交互''依赖…演化'），禁用'与''及''和'等弱并列连词；
- 主语须为明确研究对象（如'决策智能体'或'<PROJECT>中决策智能体'），逻辑动词短语须直接作用于该主语；
- 主标题须为完整主谓宾结构，无歧义缩略（如禁用'语义-时序'，须写全'语义认知与时间演化'）；
- 字数硬约束：≤26个汉字（含顿号、逗号、括号等中文标点）；理想区间：22–24字；禁用破折号、冒号、分号等非学术分隔符，仅允许空格、顿号、逗号（慎用）、括号（仅用于必要说明）；
- 每个标题须有明确逻辑主干（如'X的Y机制''Y对X的Z作用''X与Y的Z关系'），且能直接映射一个可陈述的、40字以内的核心科学问题；
- 所有标题须体现至少一项NSFC四类科学问题属性关键词：'机制''关系''原理''瓶颈''约束''驱动''稳定性''演化性'；
- 结尾统一为'研究'，不使用'探析''初探''浅析''机制探讨'等弱化表述；
- 不引入新概念或要素，仅在原要素间建立更紧密、更可证伪的关系。
