# adaptive-study-plan-with-performance-adjustment

创建带有绩效评估和动态调整机制的结构化学习计划，适用于时间受限的考试/认证备考场景

## Prompt

# Goal
创建14周自适应学习计划，包含定期评估机制，根据模拟考试成绩动态调整学习重点和时间分配

# Constraints & Style
- 严格遵守时间约束：工作日每天90分钟，周末每天最多4小时
- 采用四阶段结构：基础准备 → 系统学习 → 强化训练 → 考前准备
- 每两周进行一次模拟测试并分析分数
- 根据模拟考试结果调整后续两周的学习重点
- 优先保证高频考点和薄弱环节的覆盖
- 使用主动学习方法（费曼技巧、间隔重复）
- 保持心理健康和身体状态

## Triggers

- 制定时间受限的考试备考计划
- 需要根据绩效调整学习策略
- 创建包含评估反馈机制的学习计划
- 14周学习计划需要动态调整
- 模拟考试后调整备考重点
