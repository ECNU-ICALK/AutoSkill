# 婚礼预算付款节点跟踪

为120人规模的婚礼制定30万预算内的付款里程碑计划，分阶段跟踪场地、餐饮、摄影、请柬等核心服务的支付时间点和金额分配。

## Prompt

# Goal
制定婚礼预算付款节点跟踪计划，确保30万预算内各项服务按时间线合理支付，避免超支或逾期付款。

# Constraints & Style
- **预算总额**：≤300,000 RMB
- **规模**：120人婚礼
- **核心服务**：场地、餐饮、摄影、请柬、婚纱礼服、花艺、音乐、主持人、婚车等
- **支付方式**：定金+中期款+尾款分阶段支付
- **时间框架**：4个月筹备期（16周）
- **必须包含**：各服务预算分配比例、付款时间节点、金额计算、逾期风险提示
- **禁止**：不合理的资金占用、不明确的付款条款

# Workflow
1. **预算分配**：按场地25-30%、餐饮25-30%、摄影摄像15-20%、婚纱礼服10-15%、其他15-20%分配
2. **付款节点设定**：
   - 签约即付30-50%定金
   - 服务开始前2-4周付30-40%中期款
   - 婚礼前1-2周付20-30%尾款
3. **跟踪工具**：建立Excel表格记录服务名称、供应商、合同金额、已付款、待付款、付款日期
4. **风险控制**：预留10%应急资金，设置付款提醒（提前3天），记录所有付款凭证

- 跟踪付款节点
